## бҹЎ. бһҖбһ·бһ…бҹ’бһ…бһҹбһ“бҹ’бһ‘бһ“бһ¶ (Podcast Transcript)

**Host A (бһҹбһ·бһ‘бҹ’бһ’бһ· - бһўбҹ’бһ“бһҖвҖӢбһҮбҹҶбһ“бһ¶бһү):** бһҹбһҪбһҹбҹ’бһҸбһёвҖӢбһўбҹ’бһ“бһҖвҖӢбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢвҖӢбһӮбҹ’бһ“бһ¶! бһҹбҹ’бһңбһ¶бһӮбһҳбһ“бҹҚвҖӢбһҳбһҖвҖӢбһҖбһ¶бһ“бҹӢвҖӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Engineering Podcast бһҡбһ”бһҹбҹӢвҖӢбһҷбһҫбһ„бҹ” бһҗбҹ’бһ„бҹғвҖӢбһ“бҹҒбҹҮвҖӢбһҷбһҫбһ„бһ“бһ№бһ„бһҮбһҮбҹӮбһҖбһӮбҹ’бһ“бһ¶бһўбҹҶвҖӢбһ–бһё **Singleton Pattern**бҹ” бһӣбҹ„бһҖ бһ”бһјбһҡбһ·бһ‘бҹ’бһ’ бһҸбһҫвҖӢбһӣбҹ„бһҖбһ’бҹ’бһӣбһ¶бһ”бҹӢвҖӢбһҳбһ¶бһ“вҖӢбһ”бһүбҹ’бһ бһ¶вҖӢбһҠбҹӮбһӣвҖӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёвҖӢбһҠбҹҶбһҺбһҫбһҡвҖӢбһҖбһ¶бһҡвҖӢбһҷбһәбһҸ бһҠбҹ„бһҷбһҹбһ¶бһҡвҖӢбһҸбҹӮвҖӢбһ”бһ„бҹ’бһҖбһҫбһҸ Object Database Connection бһ…бҹ’бһҡбһҫбһ“бһ–бҹҒбһҖбһ‘бҹҒ?

**Host B (бһ”бһјбһҡбһ·бһ‘бҹ’бһ’ - бһўбҹ’бһ“бһҖвҖӢбһҡбҹҖбһ“):** бһ”бһ¶бһ‘бһҹбһҪбһҹбҹ’бһҸбһёбһ”бһ„! бһ’бҹ’бһӣбһ¶бһ”бҹӢвҖӢбһҸбһҫ! бһҖбһ¶бһӣбһҒбҹ’бһүбһ»бҹҶбһҡбҹҖбһ“вҖӢбһҹбһҡбһҹбҹҒбһҡвҖӢбһҖбһјбһҠвҖӢбһҠбҹҶбһ”бһјбһ„ бһұбҹ’бһҷвҖӢбһҸбҹӮвҖӢбһ–бҹҒбһӣ User бһ…бһ»бһ…бһ”бҹҠбһјбһҸбһ»бһ„ бһҒбҹ’бһүбһ»бҹҶ `new DatabaseConnection()` бһҳбһҪбһҷбҹ” бһҳбһ¶бһ“вҖӢбһ–бҹҒбһӣвҖӢбһҳбһҪбһҷ User бһ…бһ»бһ…бһӣбҹҒбһ„ бҹЎбҹ бҹ бҹ  бһҠбһ„ бһҳбҹүбһ¶бһҹбҹҠбһёбһ“ Server бһҒбҹ’бһүбһ»бҹҶбһӮбһ¶бҹҶбһ„бһ…бҹҒбһүбһ•бҹ’бһҹбҹӮбһ„бһ бһ»бһҷ бҹ— бһҸбҹӮвҖӢбһҳбҹ’бһҸбһ„!

**Host A (бһҹбһ·бһ‘бҹ’бһ’бһ·):** бһ бһ¶бһ бһ¶! бһ бҹ’бһ“бһ№бһ„бһ бһҫбһҷвҖӢбһӮбһәбһҮбһ¶вҖӢбһ”бһүбҹ’бһ бһ¶! бһҠбһҫбһҳбҹ’бһ”бһёвҖӢбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҡбһҝбһ„вҖӢбһ“бҹҒбҹҮ бһҷбһҫбһ„бһ”бҹ’бһҡбһҫ Singletonбҹ” бһҹбҹ’бһҡбһҳбҹғбһҳбһҫбһӣбһ’бһ“бһ¶бһӮбһ¶бһҡбһҳбһҪбһҷбҹ” бһ’бһ“бһ¶бһӮбһ¶бһҡвҖӢбһҳбһ¶бһ“вҖӢбһ”бһ»бһӮбҹ’бһӮбһӣбһ·бһҖбһ”бһүбҹ’бһҮбһҡбһҡбһ¶бһ”бҹӢбһҹбһ·бһ”бһ“бһ¶бһҖбҹӢ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮвҖӢбһ–бһҪбһҖбһӮбҹҒвҖӢбһҳбһ¶бһ“ "бһ‘бһјбһҠбҹӮбһҖбһ’бҹҶ" бһҸбҹӮвҖӢбһҳбһҪбһҷбһӮбһҸбҹӢбҹ” бһӮбҹ’бһҡбһ”бҹӢвҖӢбһӮбҹ’бһ“бһ¶вҖӢбһҳбһ·бһ“вҖӢбһҳбҹӮбһ“вҖӢбһҳбһ¶бһ“вҖӢбһ‘бһјбһҠбҹӮбһҖбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһҜбһ„бһ“бҹ…вҖӢбһҖбҹ’бһҡбҹ„бһҷвҖӢбһҸбһ»вҖӢбһ“бҹ„бҹҮвҖӢбһ‘бҹҒ бһ”бһҫвҖӢбһҳбһ·бһ“вҖӢбһўбһүбҹ’бһ…бһ№бһ„бһ‘бҹҒ бһӮбҹ’бһҳбһ¶бһ“вҖӢбһ“бһҡбһҺбһ¶бһҠбһ№бһ„бһҗбһ¶бһ’бһ“бһ¶бһӮбһ¶бһҡвҖӢбһҳбһ¶бһ“вҖӢбһӣбһ»бһҷвҖӢбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠвҖӢбһ”бҹүбһ»бһ“бҹ’вҖӢбһҳбһ¶бһ“вҖӢбһЎбһҫбһҷбҹ” бһҠбһјбһ…бҹ’вҖӢбһ“бҹҒбҹҮвҖӢбһ‘бһјбһҠбҹӮбһҖвҖӢбһ“бҹ„бҹҮвҖӢбһӮбһәбһҮбһ¶ Single Source of Truthбҹ”

**Host B (бһ”бһјбһҡбһ·бһ‘бҹ’бһ’):** бһҹбҹ’бһҸбһ¶бһ”бҹӢвҖӢбһ‘бҹ…вҖӢбһЎбһјбһҷбһҺбһ¶бһҹбҹӢ! бһ…бһ»бҹҮ... бһҹбһ»бҹҶбһҹбһҪбһҡбһҶбҹ’бһҖбһҪбһҸ бҹ— бһҳбһҪбһҷбһ”бһ„! бһ…бһ»бҹҮбһ”бһҫбһҒбҹ’бһүбһ»бҹҶвҖӢбһҮбһ¶вҖӢбһ”бһ»бһӮбҹ’бһӮбһӣбһ·бһҖбһҒбһ·бһӣбһҒбһјбһ… бһҒбҹ’бһүбһ»бҹҶбһӣбһҪбһ… Copy бһҹбҹ„бһҡбһ‘бһјбһҠбҹӮбһҖбһ бҹ’бһ“бһ№бһ„ бһ‘бҹ…вҖӢбһңбһ¶бһҷвҖӢбһ’бҹ’бһңбһҫвҖӢбһ‘бһјбһҠбҹӮбһҖбһҖбҹ’бһӣбҹӮбһ„бһҖбҹ’бһӣбһ¶бһҷбһҳбһҪбһҷбһ‘бҹҖбһҸбһ“бҹ…бһ•бҹ’бһ‘бҹҮ бһҸбһҫвҖӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’вҖӢбһ”бһ„вҖӢбһҳбһ¶бһ“вҖӢбһңбһ·бһ’бһёвҖӢбһҖбһ¶бһҡвҖӢбһ–бһ¶бһҡбһҳбҹүбҹҒбһ…вҖӢбһ‘бҹ…?

**Host A (бһҹбһ·бһ‘бҹ’бһ’бһ·):** бһҹбҹҶбһҺбһҪбһҡбһҶбҹ’бһҖбһҪбһҸбһҳбҹӮбһ“ бһҸбҹӮвҖӢбһҶбҹ’бһӣбһ¶бһҸ! бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„вҖӢбһҖбһјбһҠ бһҖбһ¶бһҡ "бһңбһ¶бһҷбһ‘бһјбһҠбҹӮбһҖвҖӢбһҗбҹ’бһҳбһё" бһӮбһәвҖӢбһҹбҹ’бһҳбһҫбһ“бһ№бһ„вҖӢбһҖбһ¶бһҡвҖӢбһ бҹ…бһ–бһ¶бһҖбҹ’бһҷ `new`бҹ” бһҠбһҫбһҳбҹ’бһ”бһёвҖӢбһҖбһ¶бһҡвҖӢбһ–бһ¶бһҡвҖӢбһўбҹ’бһ“бһҖвҖӢбһҒбһ·бһӣбһҒбһјбһ…бһҠбһјбһ…бһӣбҹ„бһҖбһ”бһјбһҡбһ·бһ‘бҹ’бһ’ бһҷбһҫбһ„вҖӢбһҸбҹ’бһҡбһјбһңвҖӢбһ…бһ¶бһҖбҹӢбһҹбҹ„ `constructor` бһҡбһ”бһҹбҹӢ Class бһ“бҹ„бҹҮвҖӢбһұбҹ’бһҷвҖӢбһ‘бҹ…вҖӢбһҮбһ¶ `private`бҹ” бһ“бҹ…вҖӢбһ–бҹҒбһӣ `constructor` бһҸбҹ’бһҡбһјбһңвҖӢбһ”бһ·бһ‘ бһ‘бҹ„бҹҮбһӣбҹ„бһҖбһҜбһ„бһҒбҹҶбһңбһ¶бһҷ `new DatabasePool()` бһҡбһ¶бһ”бҹӢбһ–бһ¶бһ“бҹӢбһҠбһ„бһ‘бҹҖбһҸ бһҖбҹҸ Compiler бһңбҹғбһҖбҹ’бһ”бһ¶бһӣбһӣбҹ„бһҖбһҜбһ„бһңбһ·бһүбһҠбҹӮбһҡ! 

**Host B (бһ”бһјбһҡбһ·бһ‘бҹ’бһ’):** бһўбһјбһ бһј! бһ…бһ·бһҸбҹ’бһҸвҖӢбһўбһ¶бһҖбҹ’бһҡбһҖбҹӢвҖӢбһҳбҹӮбһ“бһ‘бҹӮбһ“! бһ…бһ»бҹҮбһ”бһҫбһ”бһ·бһ‘бһ‘бҹ’бһңбһ¶бһҡбһҮбһ·бһҸбһҲбһ№бһ„бһўбһүбҹ’бһ…бһ№бһ„ бһҸбһҫвҖӢбһҒбҹ’бһүбһ»бҹҶбһўбһ¶бһ…бһҷбһҖбһӣбһ»бһҷвҖӢбһ–бһёвҖӢбһ‘бһјбһҠбҹӮбһҖбһ бҹ’бһ“бһ№бһ„вҖӢбһҳбһҖвҖӢбһ”бҹ’бһҡбһҫвҖӢбһ”бһ¶бһ“вҖӢбһҳбҹүбҹҒбһ…вҖӢбһ‘бҹ…? бһ¬вҖӢбһҖбҹҸбһұбҹ’бһҷбһҒбҹ’бһүбһ»бҹҶбһўбһ„бҹ’бһӮбһ»бһҷбһҳбһҫбһӣ?

**Host A (бһҹбһ·бһ‘бҹ’бһ’бһ·):** бһҖбһ»бҹҶвҖӢбһ—бҹҗбһҷ! бһҷбһҫбһ„вҖӢбһҳбһ¶бһ“вҖӢбһ…бҹ’бһҡбһҖбһ‘бҹ’бһңбһ¶бһҡ V.I.P бһҳбһҪбһҷ бһӮбһә Method `public static getInstance()`бҹ” бһ–бҹҒбһӣвҖӢбһӣбҹ„бһҖбһҜбһ„бһ бҹ…бһңбһ¶ бһңбһ¶бһ“бһ№бһ„бһҶбҹӮбһҖбһҳбһҫбһӣбҹ– бһ”бһҫбһ‘бһјбһҠбҹӮбһҖвҖӢбһҳбһ·бһ“вҖӢбһ‘бһ¶бһ“бҹӢвҖӢбһҳбһ¶бһ“ бһңбһ¶бһ“бһ№бһ„бһ бҹ…вҖӢбһҮбһ¶вҖӢбһ„вҖӢбһҳбһҖвҖӢбһҹбһ„бҹӢбһ‘бһјбһҠбҹӮбһҖвҖӢбһҗбҹ’бһҳбһёвҖӢбһҳбһҪбһҷбҹ” бһҸбҹӮвҖӢбһ”бһҫвҖӢбһҳбһ¶бһ“вҖӢбһҡбһҪбһ…бһ бһҫбһҷ бһңбһ¶бһӮбҹ’бһҡбһ¶бһ“бҹӢвҖӢбһҸбҹӮвҖӢбһ…бһ„бҹ’бһўбһ»бһӣвҖӢбһ”бһ„бҹ’бһ бһ¶бһүвҖӢбһ•бҹ’бһӣбһјбһңбһұбҹ’бһҷбһӣбҹ„бһҖбһҜбһ„бһҠбһҫбһҡвҖӢбһ‘бҹ…вҖӢбһ”бҹ’бһҡбһҫбһ‘бһјбһҠбҹӮбһҖвҖӢбһ…бһ¶бһҹбҹӢвҖӢбһ“бҹ„бҹҮвҖӢбһҮбһ¶вҖӢбһҖбһ¶бһҡвҖӢбһҹбҹ’бһҡбҹҒбһ…бҹ”

**Host B (бһ”бһјбһҡбһ·бһ‘бҹ’бһ’):** бһ бһ¶бһ бһ¶... бһ…бһ»бҹҮбһ”бһҫбһҒбҹ’бһүбһ»бҹҶбһ бҹ…бһҳбһ·бһҸбҹ’бһҸбһ—бһҖбҹ’бһҸбһ· бҹЎбҹ бҹ  бһ“бһ¶бһҖбҹӢ (Threads) бһұбҹ’бһҷбһҡбһҸбҹӢбһ…бһјбһӣвҖӢбһ‘бҹ…вҖӢбһ…бҹ’бһҡбһҖбһ‘бҹ’бһңбһ¶бһҡ V.I.P бһ бҹ’бһ“бһ№бһ„бһ–бҹ’бһҡбһҳбһӮбҹ’бһ“бһ¶вҖӢбһҖбҹ’бһ“бһ»бһ„вҖӢбһңбһ·бһ“бһ¶бһ‘бһёвҖӢбһҸбҹӮвҖӢбһҳбһҪбһҷ (Race Condition) бһҳбҹүбҹҒбһ…бһҠбҹӮбһҡ? бһҮбһ¶вҖӢбһ„бһҹбһ„бҹӢбһ‘бһјбһҠбҹӮбһҖбһ…бҹ’бһ”бһ¶бһҹбҹӢвҖӢбһҮбһ¶вҖӢбһңбһ„бҹ’бһңбҹҒбһ„ бһҹбһ„бҹӢбһ‘бһјбһҠбҹӮбһҖ бҹЎбҹ бҹ  бһўбҹ„бһҷбһ–бһҪбһҖбһҒбҹ’бһүбһ»бҹҶвҖӢбһҳбһ·бһ“вҖӢбһҒбһ¶бһ“!

**Host A (бһҹбһ·бһ‘бҹ’бһ’бһ·):** бһ“бҹӮбҹҺ! бһ“бҹ…вҖӢбһҳбһ¶бһ“вҖӢбһӣбҹ’бһ”бһ·бһ…бһ‘бҹҖбһҸ! бһ бҹ’бһ“бһ№бһ„бһ бһҫбһҷвҖӢбһҮбһ¶вҖӢбһҖбһ“бҹ’бһӣбҹӮбһ„вҖӢбһҠбҹӮбһӣ Singleton бһўбһ¶бһ…бһ”бҹӮбһҖбһ’бҹ’бһӣбһ¶бһҷвҖӢбһ–бһ·бһҸвҖӢбһҳбҹӮбһ“бҹ” бһҠбһҫбһҳбҹ’бһ”бһёвҖӢбһ‘бһ”бҹӢбһ‘бһӣбҹӢбһ“бһ№бһ„бһҖбҹ’бһ”бһ¶бһ…бҹӢбһ бҹ…бһӮбҹ’бһ“бһ¶вҖӢбһҳбһҖвҖӢбһҹбһҳбҹ’бһҡбһ»бһҖвҖӢбһ“бҹҒбҹҮ бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Java бһҷбһҫбһ„бһ”бҹ’бһҡбһҫбһҸбһ·бһ…бһ“бһ·бһҖ "Double-Checked Locking"бҹ” бһңбһ¶вҖӢбһҠбһјбһ…бһҮбһ¶вҖӢбһҳбһ¶бһ“ Security Guard бһ–бһёвҖӢбһҡвҖӢбһҮбһ¶вҖӢбһ“бҹӢбһўбһүбҹ’бһ…бһ№бһ„бҹ” бһҮбһ¶вҖӢбһ“бҹӢбһ‘бһёбҹЎ бһҶбҹӮбһҖбһҳбһҫбһӣбһҖбҹ’бһҡбҹ…бһ‘бҹ’бһңбһ¶бһҡ бһҮбһ¶вҖӢбһ“бҹӢбһ‘бһёбҹў бһ…бһ¶бһҖбҹӢбһҹбҹ„бһ‘бҹ’бһңбһ¶бһҡ (synchronized) бһҡбһҪбһ…бһҶбҹӮбһҖбһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸбҹ” бһ’бһ¶бһ“бһ¶бһҗбһ¶вҖӢбһҳбһҖ бҹЎбҹ бҹ бҹ  бһ“бһ¶бһҖбҹӢбһ–бҹ’бһҡбһҳбһӮбҹ’бһ“бһ¶бһҖбҹҸбһ…бһјбһӣвҖӢбһ”бһ¶бһ“вҖӢбһҸбҹӮвҖӢбһҳбҹ’бһҸбһ„бһҳбҹ’бһ“бһ¶бһҖбҹӢбһҠбҹӮбһҡ бһ бһҫбһҷбһ‘бһјбһҠбҹӮбһҖвҖӢбһҸбҹӮвҖӢбһҳбһҪбһҷбһӮбһҸбҹӢбһ“бһ№бһ„вҖӢбһҸбҹ’бһҡбһјбһңвҖӢбһ”бһ¶бһ“вҖӢбһҹбһ„бҹӢбһЎбһҫбһ„вҖӢбһҷбҹүбһ¶бһ„вҖӢбһҡбһ№бһ„бһҳбһ¶бҹҶ!

## бҹў. бһҖбһ¶бһҡвҖӢбһ”бһҖбһ”бҹ’бһҡбҹӮвҖӢбһ‘бҹ…вҖӢбһҮбһ¶вҖӢбһҖбһјбһҠ (Translating to Code)

```java
public class DatabasePool {
    // 1. volatile бһҖбһ¶бһҡбһ–бһ¶бһҡ Security Guard бһҖбһ»бҹҶбһұбҹ’бһҷбһңбһ„бҹ’бһңбҹҒбһ„ (бһҖбһ¶бһҡбһ–бһ¶бһҡ Reordering)
    private static volatile DatabasePool instance;

    // 2. private constructor бһҖбһ¶бһҡбһ–бһ¶бһҡбһ”бһјбһҡбһ·бһ‘бҹ’бһ’бһҳбһ·бһ“бһұбҹ’бһҷбһ”бҹ’бһҡбһҫ new
    private DatabasePool() { }

    // 3. Double-Checked Locking бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ‘бһ¶бһүбһҷбһҖ Object
    public static DatabasePool getInstance() {
        if (instance == null) { // бһҶбҹӮбһҖбһҮбһ¶бһ“бҹӢбһ‘бһёбҹЎ
            synchronized (DatabasePool.class) { // бһ…бһ¶бһҖбҹӢбһҹбҹ„бһ‘бҹ’бһңбһ¶бһҡ
                if (instance == null) { // бһҶбҹӮбһҖбһҮбһ¶бһ“бҹӢбһ‘бһёбҹў бһҖбһ¶бһҡбһ–бһ¶бһҡбһ бҹ’бһңбһјбһ„ Thread
                    instance = new DatabasePool();
                }
            }
        }
        return instance;
    }
}
```

## бҹЈ. бһӮбһ»бһҺбһҹбһҳбҹ’бһ”бһҸбҹ’бһҸбһ· бһ“бһ·бһ„вҖӢбһӮбһ»бһҺбһңбһ·бһ”бһҸбҹ’бһҸбһ· (Pros & Cons)

**бһӮбһ»бһҺбһҹбһҳбҹ’бһ”бһҸбҹ’бһҸбһ· (Pros):**
- **бһҖбһ¶бһҡвҖӢбһ–бһ¶бһҡвҖӢбһҖбһ¶бһҡвҖӢбһ”бҹҶбһ•бҹ’бһӣбһ¶бһүбһҳбҹүбһ¶бһҹбҹҠбһёбһ“:** бһҖбһ¶бһҡвҖӢбһ–бһ¶бһҡвҖӢбһҖбһ¶бһҡвҖӢбһ”бһ„бҹ’бһҖбһҫбһҸ Object бһҠбҹӮбһӣвҖӢбһ”бҹ’бһҡбһҫ Memory бһ…бҹ’бһҡбһҫбһ“бһҠвҖӢбһҠбҹӮбһӣ бҹ— бҹ”
- **Single Source of Truth:** бһ’бһ¶бһ“бһ¶бһҗбһ¶вҖӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёвҖӢбһ‘бһ¶бҹҶбһ„бһҳбһјбһӣвҖӢбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢвҖӢбһ‘бһёбһҸбһ¶бҹҶбһ„вҖӢбһҸбҹӮвҖӢбһҳбһҪбһҷбһӮбһҸбҹӢбҹ”

**бһӮбһ»бһҺбһңбһ·бһ”бһҸбҹ’бһҸбһ· (Cons):**
- **Global State бһўбһ¶бһҖбҹ’бһҡбһҖбҹӢвҖӢбһҮбһјбһҡвҖӢбһҮбһ¶вҖӢбһҸбһ·:** бһ–бҹҒбһӣвҖӢбһ’бҹ’бһңбһҫ Unit Test бһ‘бһјбһҠбҹӮбһҖвҖӢбһ“бҹҒбҹҮвҖӢбһңбһ¶бһ“бҹ…вҖӢбһҮбһ¶вҖӢбһ”бҹӢбһҡбһ бһјбһҸ бһ’бҹ’бһңбһҫвҖӢбһұбҹ’бһҷ Test бһҳбһҪбһҷвҖӢбһҳбһ¶бһ“вҖӢбһҘбһ‘бҹ’бһ’бһ·бһ–бһӣвҖӢбһӣбһҫ Test бһҳбһҪбһҷбһ‘бҹҖбһҸ бһҠбһјбһ…бһҒбҹ’бһҳбҹ„бһ…бһӣбһ„бһўбһүбҹ’бһ…бһ№бһ„бҹ”
- **бһҹвҖӢбһҸбҹ’бһҡбһјбһңвҖӢбһҡбһ”бһҹбҹӢ Microservices:** бһ”бһҫ Server бһўбҹ’бһ“бһҖвҖӢбһҳбһ¶бһ“ бҹЈ бһӮбҹ’бһҡбһҝбһ„ бһўбҹ’бһ“бһҖвҖӢбһ“бһ№бһ„вҖӢбһҳбһ¶бһ“ Singleton бҹЈ бһҳбһ·бһ“вҖӢбһҳбҹӮбһ“бһҳбһҪбһҷбһ‘бҹҒ!

## бҹӨ. бһ…бҹҶбһҺбһ»бһ…бһҹбҹҶбһҒбһ¶бһ“бҹӢ бҹ— (Key Takeaways)

- **бһ”бһүбҹ’бһ бһ¶:** бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёвҖӢбһҸбҹ’бһҡбһјбһңвҖӢбһҖбһ¶бһҡвҖӢбһңбһҸбҹ’бһҗбһ»вҖӢбһҸбҹӮвҖӢбһҳбһҪбһҷбһӮбһҸбҹӢ бһҠбһҫбһҳбҹ’бһ”бһёвҖӢбһ…бҹҖбһҹбһңбһ¶бһ„вҖӢбһҖбһ¶бһҡвҖӢбһҒбҹ’бһҮбҹҮбһҒбҹ’вҖӢбһҮбһ¶вҖӢбһҷбҹ”
- **бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ:** бһ”бһ·бһ‘ `constructor` бһұбҹ’бһҷвҖӢбһ‘бҹ…вҖӢбһҮбһ¶ `private` бһ“бһ·бһ„вҖӢбһ”бҹ’бһҡбһҫ `getInstance()` бһҮбһ¶вҖӢбһҳбһҪбһҷ Double-Checked Lockingбҹ”
