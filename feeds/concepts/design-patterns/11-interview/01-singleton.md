## бҹЎ. бһҖбһ·бһ…бҹ’бһ…бһҹбһ“бҹ’бһ‘бһ“бһ¶бһҹбһҳбҹ’бһ—бһ¶бһҹбһ“бҹҚ (Interview Transcript)

**бһўбҹ’бһ“бһҖвҖӢбһҹбһҳбҹ’бһ—бһ¶бһҹбһ“бҹҚ (бһӣбҹ„бһҖбһ”бҹ’бһҡбһ’бһ¶бһ“ - бһҸбһ№бһ„вҖӢбһҸбҹӮвҖӢбһ„ & бһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„):** бһҹбһҪбһҹбҹ’бһҸбһё бһҹбһ»бһҒбһ¶! бһҒбҹ’бһүбһ»бҹҶбһғбһҫбһүвҖӢбһҖбҹ’бһ“бһ»бһ„ CV бһ”бҹ’бһўбһјбһ“бһҠбһ¶бһҖбҹӢбһҗбһ¶ "бһҮбҹҶбһ“бһ¶бһүбһҒбһ¶бһ„ Design Patterns бһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢ"бҹ” бһўбһјбһҒбҹҒ! бһҗбҹ’бһ„бҹғвҖӢбһ“бҹҒбҹҮвҖӢбһҒбҹ’бһүбһ»бҹҶбһўбһҸбҹӢбһҹбһҪбһҡбһҡбһҝбһ„бһҹбҹ’бһҡбһҪбһӣ бҹ— бһ‘бҹҒ! бһҹбһҡбһҹбҹҒбһҡвҖӢбһҖбһјбһҠ Singleton бһҳбһҪбһҷбһ“бҹ…вҖӢбһӣбһҫвҖӢбһҖбҹ’бһҸбһ¶бһҡбһҒбҹҖбһ“бһҳбһҫбһӣбҹҚ! бһ”бһҫвҖӢбһҹбһҡбһҹбҹҒбһҡвҖӢбһҖбҹ’бһ”бһ¶бһ…бҹӢ `if (instance == null) instance = new Singleton()` бһ’бһҳбҹ’бһҳбһҸбһ¶... бһҒбҹ’бһүбһ»бҹҶбһұбҹ’бһҷбһ’бҹ’бһӣбһ¶бһҖбҹӢбһҘбһЎбһјбһңвҖӢбһ“бҹҒбҹҮ!

**бһ”бҹҒбһҖбҹ’бһҒбһҮбһ“ (бһҹбһ»бһҒбһ¶ - бһҡбһ¶бһ„бһ—бҹҗбһҷ бһҸбҹӮвҖӢбһҡбҹҖбһ”бһ…бҹҶбһҒбҹ’бһӣбһҪбһ“вҖӢбһ”бһ¶бһ“вҖӢбһӣбҹ’бһў):** бһ”бһ¶бһ‘бһӣбҹ„бһҖбһ”бҹ’бһҡбһ’бһ¶бһ“! бһҒбҹ’бһүбһ»бҹҶбһҠбһ№бһ„бһҗбһ¶вҖӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’вҖӢбһҡбһ”бһҹбҹӢвҖӢбһҖбҹ’бһҡбһ»бһҳбһ бҹҠбһ»бһ“бһҷбһҫбһ„вҖӢбһҳбһ¶бһ“вҖӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢвҖӢбһ…бҹ’бһҡбһҫбһ“ бһҠбһјбһ…бҹ’вҖӢбһ“бҹҒбҹҮвҖӢбһҖбҹ’бһ”бһ¶бһ…бҹӢбһ’бһҳбҹ’бһҳбһҸбһ¶бһ“бһ№бһ„бһ„бһ¶бһ”бҹӢвҖӢбһҠбҹ„бһҷбһҹбһ¶бһҡ Race Condition (Thread бһ…бҹ’бһҡбһҫбһ“бһ…бһјбһӣбһ–бҹ’бһҡбһҳбһӮбҹ’бһ“бһ¶)бҹ” бһҒбҹ’бһүбһ»бҹҶбһҹбһ»бҹҶвҖӢбһҹбһҡбһҹбҹҒбһҡвҖӢбһҖбҹ’бһ”бһ¶бһ…бҹӢ **Double-Checked Locking** бһҸбҹӮвҖӢбһҳбҹ’бһҸбһ„ бһҠбҹ„бһҷвҖӢбһ”бҹ’бһҡбһҫ `volatile` Keyword бһ‘бҹҖбһҸбһ•бһ„!

**бһўбҹ’бһ“бһҖвҖӢбһҹбһҳбҹ’бһ—бһ¶бһҹбһ“бҹҚ:** бһ бһ№бһҹ! бһҶбҹ’бһӣбһ¶бһҸбһӮбҹ’бһҡбһ¶бһ“бҹӢбһ”бһҫ... бһҸбҹӮвҖӢбһ”бһҫбһ”бҹ’бһўбһјбһ“бһҠбһ¶бһҖбҹӢ `private constructor` бһўбһүбҹ’бһ…бһ№бһ„ бһҒбҹ’бһүбһ»бҹҶвҖӢбһ…бһ„бҹӢвҖӢбһ”бҹ’бһҡбһ¶бһ”бҹӢбһҗбһ¶ бһҒбҹ’бһүбһ»бҹҶвҖӢбһҮбһ¶ Hacker! бһҒбҹ’бһүбһ»бҹҶбһ“бһ№бһ„бһҷбһҖ **Reflection API** бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Java бһҳбһҖвҖӢбһ‘бһҳбҹ’бһӣбһ»бҹҮбһ‘бҹ’бһңбһ¶бһҡ `private` бһ“бҹ„бҹҮ бһ бһҫбһҷ `new` бһҷбһҖ Object бһҗбҹ’бһҳбһёвҖӢбһҳбһҪбһҷбһ‘бҹҖбһҸвҖӢбһҠбҹ„бһҷвҖӢбһҖбһҳбҹ’бһӣбһ¶бҹҶбһ„бһ”бһ¶бһҷ! бһ бһ¶бһҹбһ бһ¶! бһҜбһҺбһ¶вҖӢбһ‘бҹ… Singleton бһҡбһ”бһҹбҹӢвҖӢбһ”бҹ’бһўбһјбһ“вҖӢбһ“бҹ„бҹҮ? бһ”бҹӮбһҖбһ’бҹ’бһӣбһ¶бһҷбһ”бһ¶бһҸбҹӢбһ бһҫбһҷ! 

**бһ”бҹҒбһҖбҹ’бһҒбһҮбһ“ (бһҹбһ»бһҒбһ¶ - бһ”бҹӮбһҖбһүбһҫбһҹбһҸбһҖбҹӢ бҹ— бһҸбҹӮвҖӢбһӮбһ·бһҸвҖӢбһӣбһҝбһ“):** бһўбһј! бһ”бһҫбһӣбҹ„бһҖбһ”бҹ’бһҡбһ’бһ¶бһ“бһӣбҹҒбһ„бһҖбҹ’бһ”бһ¶бһ…бҹӢ Reflection... бһҒбҹ’бһүбһ»бҹҶбһ“бһ№бһ„бһҠбһ¶бһҖбҹӢвҖӢбһўбһ“бҹ’бһ‘бһ¶бһҖбҹӢвҖӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ constructor бһҸбҹӮвҖӢбһҳбҹ’бһҸбһ„! бһҒбҹ’бһүбһ»бҹҶбһ“бһ№бһ„вҖӢбһҹбһҡбһҹбҹҒбһҡвҖӢбһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢ `if (instance != null) throw new RuntimeException("Stop hacking me!");`бҹ” бһўбһүбҹ’бһ…бһ№бһ„бһ‘бҹ„бҹҮбһӣбҹ„бһҖбһ”бҹ’бһҡбһ’бһ¶бһ“бһ‘бһҳбҹ’бһӣбһ»бҹҮбһ‘бҹ’бһңбһ¶бһҡбһ…бһјбһӣвҖӢбһҳбһҖ бһҖбҹҸвҖӢбһҸбҹ’бһҡбһјбһңвҖӢбһӮбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹӮбһҖбһҹбҹ’бһӣбһ¶бһ”бҹӢбһҒбҹ’бһӣбһҪбһ“бһҜбһ„бһҠбҹӮбһҡ!

**бһўбҹ’бһ“бһҖвҖӢбһҹбһҳбҹ’бһ—бһ¶бһҹбһ“бҹҚ (бһүбһүбһ№бһҳбһ…бһ»бһ„бһҳбһ¶бһҸбҹӢ):** бһӣбҹ’бһў! бһ…бһ»бҹҮбһ”бһҫбһҒбҹ’бһүбһ»бҹҶбһӣбҹҒбһ„бһҖбҹ’бһ”бһ¶бһ…бҹӢ **Serialization** бһңбһ·бһү? бһҒбҹ’бһүбһ»бҹҶбһҷбһҖ Object Singleton бһҡбһ”бһҹбҹӢвҖӢбһ”бҹ’бһўбһјбһ“ бһ‘бҹ…вҖӢбһ”бһҳбҹ’бһӣбҹӮбһ„вҖӢбһҮбһ¶ File бһҡбһҪбһ…бһҒбҹ’бһүбһ»бҹҶ Deserialize (бһ‘бһ¶бһүвҖӢбһ–бһё File бһҳбһҖвҖӢбһҮбһ¶ Object бһңбһ·бһү)бҹ” бһҳбҹүбһ¶бһҹбҹҠбһёбһ“ Java бһңбһ¶бһ“бһ№бһ„вҖӢбһ”бһ„бҹ’бһҖбһҫбһҸ Object бһҖбһјбһ“бһ—бҹ’бһӣбҹ„бҹҮбһҳбһҪбһҷбһ‘бҹҖбһҸвҖӢбһҠбҹ„бһҷвҖӢбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·! Singleton бһҡбһ”бһҹбҹӢвҖӢбһ”бҹ’бһўбһјбһ“бһҡбһӣбһ¶бһҷбһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸбһ бһҫбһҷ!

**бһ”бҹҒбһҖбҹ’бһҒбһҮбһ“ (бһҹбһ»бһҒбһ¶ - бһүбһүбһ№бһҳбһңбһ·бһү):** бһҒбҹ’бһүбһ»бҹҶвҖӢбһ”бһ¶бһ“вҖӢбһҸбҹ’бһҡбҹҖбһҳбһҒбҹ’бһӣбһҪбһ“бһҡбһҪбһ…бһ бһҫбһҷ! бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Java бһҠбһҫбһҳбҹ’бһ”бһёвҖӢбһ‘бһ”бҹӢбһ‘бһӣбҹӢбһ“бһ№бһ„ Serialization бһҒбҹ’бһүбһ»бҹҶбһӮбҹ’бһҡбһ¶бһ“бҹӢвҖӢбһҸбҹӮвҖӢбһҹбһҡбһҹбҹҒбһҡ Method бһңбҹҒбһ‘бһҳбһ“бҹ’бһҸбһҳбһҪбһҷбһҲбҹ’бһҳбҹ„бҹҮбһҗбһ¶ `readResolve()` бһ бһҫбһҷбһ”бҹ’бһҡбһ¶бһ”бҹӢбһңбһ¶бһұбҹ’бһҷ `return instance;` бһ…бһ¶бһҹбҹӢвҖӢбһ‘бҹ…вҖӢбһңбһ·бһүбҹ” бһ–бҹҒбһӣ Deserialize бһңбһ¶бһ“бһ№бһ„бһҡбһҸбҹӢбһ…бһјбһӣ Method бһ“бҹҒбҹҮ бһ бһҫбһҷбһ”бҹ„бҹҮбһ”бһ„бҹӢ Object бһҗбҹ’бһҳбһёвҖӢбһ…бҹ„бһӣ! 

**бһўбҹ’бһ“бһҖвҖӢбһҹбһҳбҹ’бһ—бһ¶бһҹбһ“бҹҚ (бһ‘бҹҮбһҠбҹғ):** бһўбһҹбҹ’бһ…бһ¶бһҡбҹ’бһҷбһҺбһ¶бһҹбҹӢ! бһҖбһ¶бһҡвҖӢбһ–бһ¶бһҡвҖӢбһ”бһ¶бһ“вҖӢбһ‘бһ¶бҹҶбһ„бһҖбҹ’бһ”бһ¶бһ…бҹӢ Reflection бһ‘бһ¶бҹҶбһ„бһҖбҹ’бһ”бһ¶бһ…бҹӢ Serialization бһ“бһ·бһ„ Thread-Safe бһ‘бҹҖбһҸ! бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮвҖӢбһҖбһјбһҠвҖӢбһ”бҹ’бһўбһјбһ“бһЎбһҫбһ„бһңбҹӮбһ„бһўбһ“бҹ’бһӣбһ¶бһҷ... бһ…бһ»бҹҮбһ”бһҫбһҒбҹ’бһүбһ»бҹҶбһ”бҹ’бһҡбһ¶бһ”бҹӢбһҗбһ¶ бһҳбһ¶бһ“вҖӢбһҖбҹ’бһ”бһ¶бһ…бҹӢбһҳбһҪбһҷвҖӢбһҠбҹӮбһӣвҖӢбһҹбһҡбһҹбҹҒбһҡвҖӢбһҸбҹӮвҖӢбһҳбһҪбһҷбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһӮбһҸбҹӢ бһ бһҫбһҷбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҡбһҝбһ„бһҡбһүбҹүбҹҒбһҡбһүбҹүбҹғбһ‘бһ¶бҹҶбһ„ бҹЈ бһ“бҹҒбҹҮвҖӢбһҠбҹ„бһҷвҖӢбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·вҖӢбһ–бһёвҖӢбһ’бһҳбҹ’бһҳвҖӢбһҮбһ¶вҖӢбһҸбһ· бһҸбһҫвҖӢбһ”бҹ’бһўбһјбһ“бһҠбһ№бһ„бһҗбһ¶вҖӢбһҮбһ¶вҖӢбһўбҹ’бһңбһёбһ‘бҹҒ?

**бһ”бҹҒбһҖбҹ’бһҒбһҮбһ“ (бһҹбһ»бһҒбһ¶):** бһўбһј! бһҒбҹ’бһүбһ»бҹҶбһ“бһ№бһҖбһғбһҫбһүбһ бһҫбһҷ! бһӮбһәвҖӢбһҖбһ¶бһҡвҖӢбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ **`enum`** бһҳбҹӮбһ“бһ‘бҹҒбһӣбҹ„бһҖбһ”бҹ’бһҡбһ’бһ¶бһ“? бһҹбҹҖбһңбһ—бҹ… Effective Java бһҡбһ”бһҹбҹӢвҖӢбһӣбҹ„бһҖ Joshua Bloch бһ”бһ¶бһ“вҖӢбһ”бҹ’бһҡбһ¶бһ”бҹӢбһҗбһ¶ `enum` бһӮбһәбһҮбһ¶вҖӢбһңбһ·бһ’бһёвҖӢбһҠбҹӮбһӣвҖӢбһӣбҹ’бһўвҖӢбһ“бһ·бһ„вҖӢбһҳбһ¶бһ“вҖӢбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһ”бҹҶбһ•бһ»бһҸвҖӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢвҖӢбһҖбһ¶бһҡвҖӢбһ”бһ„бҹ’бһҖбһҫбһҸ Singleton бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Java!

**бһўбҹ’бһ“бһҖвҖӢбһҹбһҳбҹ’бһ—бһ¶бһҹбһ“бҹҚ:** бһҸбҹ’бһҡбһ№бһҳвҖӢбһҸбҹ’бһҡбһјбһң! бһ”бҹ’бһўбһјбһ“вҖӢбһ–бһ·бһҸвҖӢбһҮбһ¶вҖӢбһҷбһӣбҹӢбһҠбһ№бһ„бһҹбҹҠбһёбһҮбһҳбҹ’бһҡбҹ…бһҳбҹӮбһ“ бһҳбһ·бһ“вҖӢбһҳбҹӮбһ“бһ…бҹҒбҹҮвҖӢбһҸбҹӮвҖӢбһ‘бһ“бҹ’бһ‘бҹҒбһү бҹ— бһ“бҹ„бҹҮвҖӢбһ‘бҹҒбҹ” бһўбһјбһҒбҹҒ бһҸбҹ’бһҡбҹҖбһҳбһҒбҹ’бһӣбһҪбһ“бһ”бҹ’бһҡбһҲбһҳбһҳбһ»бһҒбһ“бһ№бһ„бһҹбҹҶбһҺбһҪбһҡбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ!

## бҹў. бһҖбһјбһҠвҖӢбһҠбҹӮбһӣвҖӢбһ”бҹҒбһҖбҹ’бһҒбһҮбһ“вҖӢбһҹбһҡбһҹбҹҒбһҡ (Candidate's Code)

```java
// бһҖбҹ’бһ”бһ¶бһ…бҹӢбһ‘бһёбҹЎ: Double-Checked Locking (бһҖбһ¶бһҡбһ–бһ¶бһҡбһ”бһ¶бһ“ Thread бһҸбҹӮбһ”бҹӮбһҖбһ’бҹ’бһӣбһ¶бһҷбһҠбҹ„бһҷ Reflection & Serialization)
public class TraditionalSingleton implements Serializable {
    private static volatile TraditionalSingleton instance;

    // бһҖбһ¶бһҡбһ–бһ¶бһҡбһҖбһ¶бһҡ Hack бһҸбһ¶бһҳбһҡбһҷбҹҲ Reflection
    private TraditionalSingleton() {
        if (instance != null) {
            throw new RuntimeException("Stop hacking me with Reflection!");
        }
    }

    public static TraditionalSingleton getInstance() {
        if (instance == null) {
            synchronized (TraditionalSingleton.class) {
                if (instance == null) {
                    instance = new TraditionalSingleton();
                }
            }
        }
        return instance;
    }

    // бһҖбһ¶бһҡбһ–бһ¶бһҡбһҖбһ¶бһҡ Hack бһҸбһ¶бһҳбһҡбһҷбҹҲ Serialization
    protected Object readResolve() {
        return getInstance();
    }
}

// ==========================================

// бһҖбҹ’бһ”бһ¶бһ…бҹӢбһ‘бһёбҹў: Enum Singleton (бһҖбҹ’бһ”бһ¶бһ…бҹӢбһҖбҹҶбһ–бһјбһӣбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһ—бһ¶бһҹбһ“бҹҚбһ…бһ„бҹӢбһғбһҫбһү)
// бһҖбһ¶бһҡбһ–бһ¶бһҡбһҡбһ¶бһӣбҹӢбһ”бһүбҹ’бһ бһ¶бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ–бһёбһ’бһҳбҹ’бһҳбһҮбһ¶бһҸбһ·бһҡбһ”бһҹбҹӢ JVM!
public enum UltimateSingleton {
    INSTANCE; // бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶ Singleton Object бһҸбҹӮбһҳбһҪбһҷбһӮбһҸбҹӢ!

    public void doSomething() {
        System.out.println("Enum Singleton is bulletproof!");
    }
}
```

## бҹЈ. бһҖбһ¶бһҡвҖӢбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғвҖӢбһҡбһ”бһҹбҹӢвҖӢбһўбҹ’бһ“бһҖвҖӢбһҹбһҳбҹ’бһ—бһ¶бһҹбһ“бҹҚ (Interviewer's Evaluation)

вң… **бһ…бҹҶбһҺбһ»бһ…вҖӢбһҒбҹ’бһӣбһ¶бҹҶбһ„вҖӢбһҡбһ”бһҹбҹӢвҖӢбһ”бҹҒбһҖбҹ’бһҒбһҮбһ“:**
- бһҷбһӣбҹӢбһҠбһ№бһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢвҖӢбһ–бһёвҖӢбһ”бһүбҹ’бһ бһ¶ Race Condition бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„вҖӢбһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“ Multi-threading (бһ…бҹҒбҹҮбһ”бҹ’бһҡбһҫ `volatile` бһ“бһ·бһ„ `synchronized`)бҹ”
- бһҳбһ·бһ“вҖӢбһҸбһҖбҹӢбһҹбҹ’бһӣбһ»бһҸбһ“бһ№бһ„бһҹбҹҒбһҺбһ¶бһҡбһёбһҷбҹүбһјбһ”бҹҶбһ•бҹ’бһӣбһ·бһ…бһ”бҹҶбһ•бҹ’бһӣбһ¶бһү (Hacking scenarios) бһҠбһјбһ…бһҮбһ¶ Reflection бһ“бһ·бһ„ Serializationбҹ”
- бһҹбҹ’бһӮбһ¶бһӣбҹӢ Best Practices бһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷ (бһҖбһ¶бһҡвҖӢбһ”бҹ’бһҡбһҫ `enum` бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Singleton)бҹ”

вқҢ **бһ…бҹҶбһҺбһ»бһ…бһҒбҹ’бһҹбҹ„бһҷ (Trap Question):**
- бһ”бҹҒбһҖбҹ’бһҒбһҮбһ“бһӮбһҪбһҡвҖӢбһҸбҹӮвҖӢбһҡбҹҶбһӣбһ№бһҖвҖӢбһ–бһёвҖӢбһ”бһүбҹ’бһ бһ¶ Anti-Pattern бһҡбһ”бһҹбҹӢ Singleton бһ•бһ„бһҠбҹӮбһҡ (бһҠбһјбһ…бһҮбһ¶вҖӢбһңбһ¶вҖӢбһ’бҹ’бһңбһҫвҖӢбһұбҹ’бһҷвҖӢбһҖбһјбһҠвҖӢбһ–бһ·бһ”бһ¶бһҖвҖӢбһ’бҹ’бһңбһҫ Unit Test бһ–бҹ’бһҡбҹ„бҹҮвҖӢбһңбһ¶вҖӢбһҮбһ¶ Global State)бҹ”

**бһӣбһ‘бҹ’бһ’бһ•бһӣ:** бһҶбҹ’бһӣбһ„бһҖбһ¶бһҸбҹӢ! (Pass) бһўбҹ’бһ“бһҖвҖӢбһ–бһ·бһҸвҖӢбһҮбһ¶вҖӢбһҹбҹҗбһҖбҹ’бһҸбһ·бһҹбһҳвҖӢбһҹбһҡбһҹбҹҒбһҡвҖӢбһҖбһјбһҠвҖӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ Core Banking бһҡбһ”бһҹбҹӢвҖӢбһҖбҹ’бһҡбһ»бһҳбһ бҹҠбһ»бһ“бһҷбһҫбһ„!
