## бҹЎ. бһ”бһүбҹ’бһ бһ¶бһҹбҹ’бһ“бһјбһӣ (The Core Problem)

бһҒбҹ’бһүбһ»бҹҶбһҹбһјбһҳбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҠбҹ„бһҷбһҹбҹҶбһҺбһҪбһҡбһҳбһҪбһҷбҹ” бһҹбҹ’бһҡбһҳбҹғбһҗбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҳбһ¶бһ“ printer бһҸбҹӮбһҳбһҪбһҷ бһҳбһ¶бһ“ database connection pool бһҸбҹӮбһҳбһҪбһҷ бһ¬бһҳбһ¶бһ“бһҜбһҖбһҹбһ¶бһҡ config бһҠбҹӮбһӣбһ•бҹ’бһ‘бһ»бһҖбһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҳбҹүбһјбһҡбһёбһҸбҹӮбһҳбһҪбһҷбҹ” бһ•бҹ’бһ“бҹӮбһҖбһҳбһҪбһҷбһ“бҹғбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһңбһ¶ бһ бһҫбһҷбһҳбһҪбһҷбһ—бҹ’бһӣбҹӮбһҸбһҖбҹ’бһҡбҹ„бһҷбһҳбһҖ вҖ” бһ“бҹ…бһҖбһ“бҹ’бһӣбҹӮбһ„бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһ‘бһ¶бҹҶбһ„бһҹбҹ’бһҡбһ»бһ„ вҖ” бһ•бҹ’бһ“бҹӮбһҖбһҳбһҪбһҷбһ‘бҹҖбһҸбһҖбҹҸбһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһңбһ¶бһҠбҹӮбһҡбҹ” бһҸбһҫбһўбҹ’бһ“бһҖбһ’бһ¶бһ“бһ¶бһҠбҹ„бһҷбһҡбһ”бҹҖбһ”бһҺбһ¶бһҗбһ¶ бһ•бҹ’бһ“бҹӮбһҖбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһ“бҹҒбҹҮбһҖбҹҶбһ–бһ»бһ„бһ“бһ·бһҷбһ¶бһҷбһўбҹҶбһ–бһёбһңбһҸбҹ’бһҗбһ» *бһҠбһјбһ…бһӮбҹ’бһ“бһ¶* бһҳбһ·бһ“бһҳбҹӮбһ“бһ…бҹ’бһ”бһ¶бһ”бҹӢбһ…бһҳбҹ’бһӣбһ„бһ–бһёбһҡ бһҠбҹӮбһӣбһ”бһ¶бһ“бһүбҹӮбһҖбһ…бҹҒбһүбһ–бһёбһӮбҹ’бһ“бһ¶бһҠбҹ„бһҷбһҹбҹ’бһ„бһ¶бһҸбҹӢбҹ—?

бһҹбһјбһҳбһ…бһ„бһ…бһ¶бҹҶбһҹбҹҶбһҺбһҪбһҡбһ“бҹҒбҹҮбһ‘бһ»бһҖ бһ–бҹ’бһҡбҹ„бҹҮбһ”бһүбҹ’бһ бһ¶бһ“бҹҒбҹҮбһҳбһ¶бһ“бһ—бһ¶бһ–бһӣбһ¶бһҖбҹӢбһҖбҹҶбһ”бһ¶бҹҶбһ„бҹ” бһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһҖбһјбһҠбһ–бһёбһҡбһ•бҹ’бһ“бҹӮбһҖбһ”бһ„бҹ’бһҖбһҫбһҸбһ…бҹ’бһ”бһ¶бһ”бҹӢбһ…бһҳбҹ’бһӣбһ„бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһҡбһ”бһҹбҹӢ В«theВ» connection pool бһ“бҹ„бҹҮ бһӮбҹ’бһҳбһ¶бһ“бһўбҹ’бһңбһёбһҒбһјбһ…бһ—бҹ’бһӣбһ¶бһҳбҹ—бһЎбһҫбһҷбҹ” бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ” Test бһҖбҹҸбһҮбһ¶бһ”бҹӢбҹ” бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ production бһҖбҹ’бһҡбҹ„бһҳбһ”бһ“бҹ’бһ‘бһ»бһҖбһ’бҹ’бһ„бһ“бҹӢ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҮбһҪбһ”бһ“бһјбһң bug бһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһҹбһ”бҹ’бһҸбһ¶бһ бҹҚбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһңбһ№бһҖбһңбһҡбҹ– бһҳбҹҒбһҳбҹүбһјбһҡбһёбһ бһјбһҡбһ бҹҖбһҡ (Memory Exhaustion) port hardware бһҮбһ¶бһ”бҹӢбһӮбһ¶бҹҶбһ„ бһ¬ cache бһ–бһёбһҡбһҠбҹӮбһӣбһ•бҹ’бһ‘бһ»бһҷбһӮбҹ’бһ“бһ¶бһўбҹҶбһ–бһёбһҖбһ¶бһҡбһ–бһ·бһҸбҹ” бһӮбҹ’бһҳбһ¶бһ“бһ“бһҡбһҺбһ¶бһҹбһҡбһҹбҹҒбһҡ bug бһ“бҹҒбҹҮбһ‘бҹҒбҹ” bug бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶ bug *бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ (structural)* вҖ” бһңбһ¶бһҖбһҫбһҸбһ…бҹҒбһүбһ–бһёбһҖбһ¶бһҡбһ–бһ·бһҸбһҠбҹӮбһӣбһҗбһ¶ бһ—бһ¶бһҹбһ¶бһҹбһҡбһҹбҹҒбһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ¶бһ“бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһ…бҹ’бһ”бһ¶бһ”бҹӢбһ…бһҳбҹ’бһӣбһ„бһ–бһёбһҡбһўбһ¶бһ…бһҳбһ¶бһ“бһңбһҸбҹ’бһҸбһҳбһ¶бһ“бһҸбһ¶бҹҶбһ„бһ–бһёбһҠбҹҶбһ”бһјбһ„бҹ”

бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮ бһҷбһҫбһ„бһ…бһ„бҹӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ’бһ“бһ’бһ¶бһ“бһҡбһҪбһҳбһӮбҹ’бһ“бһ¶ (бһҠбһјбһ…бһҮбһ¶ Database Connection Pool бһ¬ Hardware Port) бһҠбҹӮбһӣбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸ Object бһ…бҹ’бһҡбһҫбһ“бһ“бһ№бһ„бһ”бһҺбҹ’бһҸбһ¶бһӣбһұбҹ’бһҷбһ бһјбһҡбһ бҹҖбһҡбһҳбҹҒбһҳбҹүбһјбһҡбһё (Memory Exhaustion) бһҖбһ¶бһҡбһҮбһ¶бһ”бҹӢбһӮбһ¶бҹҶбһ„бһҒбҹ’бһҹбҹӮбһҹбҹ’бһҡбһЎбһ¶бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ (Thread Lockups) бһ¬бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбһ·бһ“бһҹбҹҠбһёбһҹбһ„бҹ’бһңбһ¶бһҖбҹӢбһӮбҹ’бһ“бһ¶ (Inconsistent State)бҹ”

## бҹў. бһҖбһ¶бһҡбһ‘бһ¶бһүбһ бҹҒбһҸбһ»бһ•бһӣбһ–бһёбһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһӮбҹ’бһҡбһ№бҹҮ (First Principles Derivation)

бһҷбһҫбһ„бһҖбһ»бҹҶбһ‘бһ“бҹ’бһ‘бҹҒбһүбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһЎбһҫбһҷбҹ” бһ…бһјбһҡбһҷбһҫбһ„ *бһ‘бһ¶бһүбһңбһ¶бһ…бҹҒбһү* бһҠбҹ„бһҷбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ–бһ·бһҸбһҮбһ¶бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһ бһҫбһҷбһҳбһҫбһӣбһҗбһ¶ бһҸбһҫ pattern бһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҒбҹҶбһұбҹ’бһҷбһҖбһҫбһҸбһЎбһҫбһ„бһӣбһҫбһҷбһҫбһ„бһ¬бһўбһҸбҹӢбҹ”

**бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ–бһёбһҖбһ¶бһҡбһ–бһ·бһҸбһҳбһҪбһҷбһўбҹҶбһ–бһёбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бҹ”** бһҡбһ¶бһӣбҹӢбһ–бҹҒбһӣбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҹбһҡбһҹбҹҒбһҡ `new` бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҖбһ¶бһҸбҹӢбһҷбһҖбһ”бҹ’бһӣбһ»бһҖбһҳбҹҒбһҳбҹүбһјбһҡбһёбһҗбҹ’бһҳбһёбһҳбһҪбһҷ бһҠбҹӮбһӣбһҠбһ¶бһ…бҹӢбһҠбҹ„бһҷбһЎбҹӮбһҖбһ–бһёбһӮбҹ’бһ“бһ¶бһ‘бһ¶бҹҶбһ„бһҹбҹ’бһҡбһ»бһ„бҹ” бһ“бҹҒбҹҮбһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹбһҡбһ…бһ“бһ¶бһ”бһҗбһЎбһҫбһҷ вҖ” бһңбһ¶бһҮбһ¶ *бһўбһҸбҹ’бһҗбһ“бҹҗбһҷ* бһҡбһ”бһҹбҹӢбһ–бһ¶бһҖбҹ’бһҷ `new`бҹ” бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮ бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһӮбһә В«бһҸбҹ’бһҡбһјбһңбһҳбһ¶бһ“бһңбһҸбҹ’бһҗбһ»бһ“бҹҒбҹҮ бһҸбҹӮбһҳбһҪбһҷбһӮбһҸбҹӢВ» бһ“бҹ„бҹҮбһҡбһ¶бһӣбҹӢбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫ `new` бһҠбҹ„бһҷбһҡбһҹбһ¶бһҸбҹӢ бһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһӮбҹҶбһҡбһ¶бһҳбһҖбҹҶбһ бҹӮбһ„бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҠбһӣбҹӢбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ“бҹ„бҹҮбҹ” бһӮбҹ’бһҡбҹ„бҹҮбһҗбҹ’бһ“бһ¶бһҖбҹӢбһ“бҹҒбҹҮбһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶бһҡбһҝбһ„бһҹбҹ’бһҳбһ¶бһ“бһ“бҹ„бҹҮбһ‘бҹҒ вҖ” бһңбһ¶бһ”бһ¶бһ“бһҖбһ”бҹӢбһҒбҹ’бһӣбһҪбһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ–бһ¶бһҖбҹ’бһҷбһӮбһ“бҹ’бһӣбһ№бҹҮбһ“бҹҒбҹҮбһҸбҹӮбһҳбҹ’бһҸбһ„бҹ”

**бһҘбһЎбһјбһңбһ“бҹҒбҹҮ бһҹбһјбһҳбһҹбһҪбһҡбһҹбҹҶбһҺбһҪбһҡбһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһҷбһҫбһ„бһҳбһ·бһ“бһҹбҹ’бһҡбһҪбһӣбһҖбҹ’бһ“бһ»бһ„бһ…бһ·бһҸбҹ’бһҸбҹ”** бһҸбһҫбһҳбһ¶бһ“бһўбҹ’бһңбһёбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ“бҹҒбҹҮ бһҠбҹӮбһӣбһҡбһ¶бһҡбһ¶бҹҶбһ„ developer бһҺбһ¶бһҳбҹ’бһ“бһ¶бһҖбҹӢ бһ“бҹ…бһҖбһ“бҹ’бһӣбҹӮбһ„бһҺбһ¶бһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„ codebase бһҳбһ·бһ“бһұбҹ’бһҷбһңбһ¶бһҷ `new ConnectionPool()` бһҮбһ¶бһӣбһҫбһҖбһ‘бһёбһ–бһёбһҡ? бһ“бһ·бһҷбһ¶бһҷбһұбҹ’бһҷбһҸбҹ’бһҡбһ„бҹӢбһ‘бҹ… вҖ” бһ…бһҳбҹ’бһӣбһҫбһҷбһӮбһә *бһӮбҹ’бһҳбһ¶бһ“бһўбҹ’бһңбһёбһ‘бһ¶бһӣбҹӢбһҸбҹӮбһҹбҹ„бҹҮ*бҹ” Constructor бһӮбһә public бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһ‘бҹ’бһңбһ¶бһҡбһ“бҹ…бһ”бһҫбһҖбһ…бҹҶбһ бҹ” бһҠбһҡбһ¶бһ”бһҺбһ¶бһ‘бҹ’бһңбһ¶бһҡбһ“бҹ„бҹҮбһ“бҹ…бһ”бһҫбһҖбһ…бҹҶбһ  бһӮбҹ’бһҳбһ¶бһ“бһңбһ·бһ“бҹҗбһҷ бһ¬бһҜбһҖбһҹбһ¶бһҡбһҺбһ¶бһҳбһҪбһҷбһўбһ¶бһ…бһҹбһ„бҹ’бһӮбҹ’бһҡбҹ„бҹҮбһҷбһҫбһ„бһ”бһ¶бһ“бһЎбһҫбһҷ вҖ” бһ“бһ№бһ„бһҳбһ¶бһ“бһ“бһҡбһҺбһ¶бһҳбҹ’бһ“бһ¶бһҖбҹӢ бһ“бҹ…бһҗбҹ’бһ„бҹғбһҺбһ¶бһҳбһҪбһҷ бһҠбһҫбһҡбһ…бһјбһӣбһҸбһ¶бһҳбһ‘бҹ’бһңбһ¶бһҡбһ“бҹ„бҹҮбһҮбһ¶бһҳбһ·бһ“бһҒбһ¶бһ“бҹ”

**бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮ бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ“бҹҒбҹҮбһҖбҹҸбһҹбһҡбһҹбҹҒбһҡбһҒбҹ’бһӣбһҪбһ“бһңбһ¶бһЎбһҫбһ„бһңбһ·бһүбһҠбҹ„бһҷбһҹбҹ’бһ„бһ¶бһҸбҹӢбҹ—бҹ”** В«бһ’бһ¶бһ“бһ¶бһұбҹ’бһҷбһҳбһ¶бһ“ instance бһҸбҹӮбһҳбһҪбһҷВ» бһҸбһ¶бһҳбһ–бһ·бһҸбһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶ В«бһҠбһҖбһ…бҹҒбһүбһ–бһёбһўбҹ’бһ“бһҖбһҠбһ‘бҹғбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ“бһјбһң *бһўбҹҶбһҺбһ¶бһ…* бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸ instanceВ»бҹ” бһҷбһҫбһ„бһҳбһ·бһ“бһҹбһ»бҹҶбһұбҹ’бһҷбһӮбҹҒбһҖбһ»бҹҶбһ бҹ… `new` бһЎбһҫбһҷ вҖ” бһҷбһҫбһ„бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһңбһ¶бһҳбһ·бһ“бһўбһ¶бһ…бһ‘бҹ…бһҡбһҪбһ…бҹ” бһҷбһҫбһ„бһ…бһ¶бһҖбҹӢбһҹбҹ„ Constructorбҹ– бһ”бҹ’бһҸбһјбһҡбһңбһ¶бһ‘бҹ…бһҮбһ¶ `private`бҹ”

**бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҘбһЎбһјбһңбһ“бҹҒбҹҮ бһҷбһҫбһ„бһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһүбҹ’бһ бһ¶бһҗбҹ’бһҳбһёбһҳбһҪбһҷ бһҳбҹӮбһ“бһ‘бҹҒ?** бһ”бһҫбһ‘бҹ’бһңбһ¶бһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ…бһ¶бһҖбҹӢбһҹбҹ„ бһҸбһҫбһ“бһҡбһҺбһ¶бһўбһ¶бһ…бһ‘бһ¶бһүбһҷбһҖ Object бһ“бҹ„бҹҮбһҳбһҖбһ”бҹ’бһҡбһҫбһ”бһ¶бһ“? бһҷбһҫбһ„бһ”бһ¶бһ“бһҖбһ¶бһҡбһ–бһ¶бһҡбһ’бһ“бһ’бһ¶бһ“бһ“бҹҒбҹҮбһҷбҹүбһ¶бһ„бһҮбһ·бһҸбһҹбҹ’бһ“бһ·бһ‘бҹ’бһ’ бһҡбһ бһјбһҸбһҠбһӣбҹӢбһӮбҹ’бһҳбһ¶бһ“бһ“бһҡбһҺбһ¶бһҳбҹ’бһ“бһ¶бһҖбҹӢбһўбһ¶бһ…бһ…бһјбһӣбһҠбһӣбҹӢбһңбһ¶бһ”бһ¶бһ“бҹ” бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ“бҹҒбҹҮбһ бһҫбһҷ бһҮбһ¶бһӮбһ“бҹ’бһӣбһ№бҹҮбһ‘бһ¶бҹҶбһ„бһҹбҹ’бһҡбһ»бһ„бҹ– бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫ Class бһ“бҹ„бҹҮ бһҮбһ¶бһўбҹ’бһ“бһҖбһҸбҹӮбһҳбһҪбһҷбһӮбһҸбҹӢбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһ”бһ„бҹ’бһҖбһҫбһҸбһҒбҹ’бһӣбһҪбһ“бһҜбһ„ бһ“бҹ„бҹҮ *Class бһ“бҹ„бҹҮбһ•бҹ’бһ‘бһ¶бһӣбҹӢ* бһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҮбһ¶бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһӮбһӣбҹӢ instance бһ…бҹҒбһүбҹ” бһҷбһҫбһ„бһ”бһ“бҹ’бһҗбҹӮбһҳбһ…бҹ’бһҡбһҖбһ‘бҹ’бһңбһ¶бһҡбһҠбҹӮбһӣбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһ¶бһ“бһҳбһҪбһҷ вҖ” бһӮбһәбһҳбһ»бһҒбһ„бһ¶бһҡ `public static` бһҲбҹ’бһҳбҹ„бҹҮ `getInstance()` вҖ” бһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ„бһ¶бһҡбһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶ бһҠбҹӮбһӣбһӮбҹ’бһҳбһ¶бһ“бһ“бһҡбһҺбһ¶бһ•бҹ’бһҹбҹҒбһ„бһўбһ¶бһ…бһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбһ”бһ¶бһ“бҹ– бһ–бҹҒбһӣбһ бҹ…бһӣбһҫбһҖбһҠбҹҶбһ”бһјбһ„ бһңбһ¶бһ”бһ„бҹ’бһҖбһҫбһҸ instance бһҸбҹӮбһҳбһҪбһҷбһ“бҹ„бҹҮ; бһҡбһ¶бһӣбҹӢбһ–бҹҒбһӣбһҖбҹ’бһҡбҹ„бһҷбҹ—бһҳбһҖ бһңбһ¶бһ”бҹ’бһҡбһӮбһӣбҹӢ instance бһҠбһҠбҹӮбһӣбһ“бҹ„бҹҮбһҸбҹ’бһҡбһЎбһ”бҹӢбһ‘бҹ…бһңбһ·бһүбҹ”

бһ бһҫбһҷбһ“бҹҒбҹҮбһ бһҫбһҷ бһӮбһәбһҮбһ¶ pattern бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“ *бһ‘бһ¶бһүбһ…бҹҒбһү* бһҳбһ·бһ“бһҳбҹӮбһ“бһ‘бһ“бҹ’бһ‘бҹҒбһүбҹ– бһ…бһ¶бһҖбҹӢбһҹбҹ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸ бһҡбһҪбһ…бһ”бһҫбһҖбһ…бҹ’бһҡбһҖбһ‘бҹ’бһңбһ¶бһҡбһҸбҹӮбһҳбһҪбһҷ бһҠбҹӮбһӣбһ’бһ¶бһ“бһ¶бһ—бһ¶бһ–бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бҹ” бһҹбһјбһҳбһҖбһҸбҹӢбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҗбһ¶ бһҷбһҫбһ„бһҳбһ·бһ“бһҠбҹӮбһӣ *бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ* Singleton бһЎбһҫбһҷ вҖ” бһўбһ¶бһҖбһ”бҹ’бһ”бһҖбһ·бһҡбһ·бһҷбһ¶бһҡбһ”бһҹбҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“ бһ”бһјбһҖбһ“бһ№бһ„бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҡбһ”бһҹбҹӢбһҷбһҫбһ„ бһ”бһ¶бһ“бһ‘бһ»бһҖбһұбҹ’бһҷбһҷбһҫбһ„бһӮбҹ’бһҳбһ¶бһ“бһ‘бҹ’бһңбһ¶бһҡбһ•бҹ’бһҹбҹҒбһ„бһҺбһ¶бһҳбһҪбһҷбһЎбһҫбһҷбҹ”

## бҹЈ. бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһҖбһјбһҠбһӮбҹҶбһҡбһј (Code Architecture)

бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ“бҹҒбҹҮ бһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһ‘бһ¶бһүбһ бҹҒбһҸбһ»бһ•бһӣбһҡбһ”бһҹбҹӢбһҷбһҫбһ„ бһ”бһҖбһ”бҹ’бһҡбҹӮбһ‘бҹ…бһҮбһ¶ Javaбҹ– бһ…бһ¶бһҖбҹӢбһҹбҹ„ Constructor бһҡбһҪбһ…бһ”бһҫбһҖбһ…бҹ’бһҡбһҖ `getInstance()` бһҸбҹӮбһҳбһҪбһҷ бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһ…бҹҶбһ–бҹ„бҹҮ threadбҹ” бһҷбһҫбһ„бһ”бҹ’бһҡбһҫ **double-checked locking** бһҮбһ¶бһҳбһҪбһҷ `volatile` бһҠбһҫбһҳбҹ’бһ”бһёбһұбҹ’бһҷ thread бһ…бҹ’бһҡбһҫбһ“бһҳбһ·бһ“бһ”бһ„бҹ’бһҖбһҫбһҸ instance бһ…бҹ’бһҡбһҫбһ“бһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһҸбҹӮбһҳбһҪбһҷбҹ–

```java
public final class ConnectionPool {

    // volatile бһ’бһ¶бһ“бһ¶бһҗбһ¶ thread бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһғбһҫбһү instance бһҠбһјбһ…бһӮбҹ’бһ“бһ¶ (бһҳбһ·бһ“бһҳбһ¶бһ“ cache бһҒбһјбһҹ)
    private static volatile ConnectionPool instance;

    // бһ…бһ¶бһҖбҹӢбһҹбҹ„бһ‘бҹ’бһңбһ¶бһҡбҹ– бһӮбҹ’бһҳбһ¶бһ“бһ“бһҡбһҺбһ¶бһўбһ¶бһ…бһ бҹ… new ConnectionPool() бһ–бһёбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ…бһ”бһ¶бһ“бһЎбһҫбһҷ
    private ConnectionPool() {
        if (instance != null) {
            throw new IllegalStateException("бһ”бҹ’бһҡбһҫ getInstance() бһҮбҹҶбһ“бһҪбһҹбһңбһ·бһү");
        }
        // ... initialize the pool ...
    }

    // бһ…бҹ’бһҡбһҖбһ‘бҹ’бһңбһ¶бһҡбһҸбҹӮбһҳбһҪбһҷбһӮбһҸбҹӢ бһҠбҹӮбһӣбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһ¶бһ“
    public static ConnectionPool getInstance() {
        if (instance == null) {                 // бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһӣбһҫбһҖбһ‘бһё бҹЎ (бһҳбһ·бһ“бһ…бһ¶бһҖбҹӢбһҹбҹ„ вҖ” бһӣбһҝбһ“)
            synchronized (ConnectionPool.class) {
                if (instance == null) {         // бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһӣбһҫбһҖбһ‘бһё бҹў (бһ…бһ¶бһҖбҹӢбһҹбҹ„ вҖ” бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–)
                    instance = new ConnectionPool();
                }
            }
        }
        return instance;
    }
}
```

**бһҖбҹҶбһҺбһҸбҹӢбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢ вҖ” бһңбһ·бһ’бһёбһҹбһ¶бһҳбһүбҹ’бһү бһ“бһ·бһ„бһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҖбҹ’бһ“бһ»бһ„ Javaбҹ–** бһҸбһ¶бһҳбһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһҖбҹ’бһ“бһ»бһ„бһҹбҹҖбһңбһ—бҹ… *Effective Java* бһңбһ·бһ’бһёбһҠбҹҸбһҡбһ№бһ„бһҳбһ¶бҹҶбһ”бҹҶбһ•бһ»бһҸбһӮбһәбһ”бҹ’бһҡбһҫ `enum` бһҠбҹӮбһӣбһҳбһ¶бһ“бһ’бһ¶бһҸбһ»бһҸбҹӮбһҳбһҪбһҷбҹ” JVM бһ’бһ¶бһ“бһ¶бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– thread бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҸбҹӮбһҳбҹ’бһҸбһ„ бһ“бһ·бһ„бһҖбһ¶бһҡбһҖбһ¶бһҡбһ–бһ¶бһҡ reflection/serialization бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бҹ–

```java
public enum ConnectionPool {
    INSTANCE;

    public void query(String sql) {
        // ... бһ”бҹ’бһҡбһҫ ConnectionPool.INSTANCE.query(...) ...
    }
}
```

`enum` бһҒбҹ’бһӣбһёбһҮбһ¶бһ„ бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҮбһ¶бһ„ бһ бһҫбһҷбһӣбһ»бһ”бһ”бҹҶбһ”бһ¶бһҸбҹӢбһҡбһ¶бһӣбҹӢбһҖбһ¶бһҡбһӣбҹҶбһ”бһ¶бһҖбһ“бҹғ double-checked lockingбҹ” бһ”бҹ’бһҡбһҫ DCL бһҸбҹӮбһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҖбһ¶бһҡ initialize бһҠбҹҸбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү бһҠбҹӮбһӣ enum бһҳбһ·бһ“бһўбһ¶бһ…бһ”бһ„бҹ’бһ бһ¶бһүбһ”бһ¶бһ“бһҹбҹ’бһҡбһҪбһӣбҹ”
