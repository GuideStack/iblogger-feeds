## бһӣбҹҶбһ“бһ¶бҹҶбһ”бһүбҹ’бһ бһ¶ (The Pattern)

бһҸбһҫбһўбҹ’бһ“бһҖбһ’бҹ’бһӣбһ¶бһ”бҹӢбһ…бһјбһӣбһ‘бҹ…бһҖбҹӮбһ”бҹ’бһҡбҹӮ Code бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһӮбһҳбҹ’бһҡбҹ„бһ„бһ…бһ¶бһҹбҹӢ (Legacy Project) бһҳбһҪбһҷ бһ бһҫбһҷбһҹбҹ’бһҡбһ¶бһ”бҹӢбһҸбҹӮбһҳбһ¶бһ“бһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһ—бҹҗбһҷбһҒбҹ’бһӣбһ¶бһ… бһ бһ¶бһҖбҹӢбһҠбһјбһ…бһҮбһ¶бһҖбҹҶбһ–бһ»бһ„бһҠбһҫбһҡбһ…бһјбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҡбһјбһ„бһ—бҹ’бһ“бҹҶбһҠбҹҸбһ„бһ„бһ№бһҸбһҠбҹӮбһӣбһ–бҹ„бһҡбһ–бҹҒбһүбһҠбҹ„бһҷбһ•бҹ’бһӣбһјбһңбһҒбҹ’бһңбһ¶бһҸбҹӢбһҒбҹ’бһңбҹӮбһ„ бһ“бһ·бһ„бһҡбһҖбһ…бҹ’бһҡбһҖбһ…бҹҒбһүбһҳбһ·бһ“бһғбһҫбһүбһҠбҹӮбһҡбһ¬бһ‘бҹҒ? 

бһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһўбҹ’бһ“бһҖбһҖбҹӮбһ”бҹ’бһҡбҹӮбһҖбһјбһҠбһҸбҹӮбһҳбһҪбһҷбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ“бҹ…бһҖбһ“бҹ’бһӣбҹӮбһ„бһ“бҹҒбҹҮ бһҹбҹ’бһҡбһ¶бһ”бҹӢбһҸбҹӮбһҳбһ¶бһ“бһҖбҹҶбһ бһ»бһҹ (Error/Bug) бһӣбҹ„бһҸбһ…бҹҒбһүбһҳбһҖбһ“бҹ…бһҖбһ“бҹ’бһӣбҹӮбһ„бһ•бҹ’бһҹбҹҒбһ„бһҳбһҪбһҷбһ‘бҹҖбһҸ бһҠбҹӮбһӣбһҳбһҫбһӣбһ‘бҹ…бһҳбһ·бһ“бһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бһӮбҹ’бһ“бһ¶бһ‘бһ¶бһӣбҹӢбһҸбҹӮбһҹбҹ„бҹҮбҹ” 

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһңбһ·бһҹбҹҗбһҷбһңбһ·бһҹбҹ’бһңбһҖбһҳбҹ’бһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё (Software Engineering) бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһӮбҹҒбһ бҹ…бһҗбһ¶ **Spaghetti Code** (бһҖбһјбһҠбһҠбҹӮбһӣбһҮбҹҶбһ–бһ¶бһҖбҹӢбһңбһ¶бһҖбҹӢбһңбһ·бһ“бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһҠбһјбһ…бһҹбһҡбһҹбҹғбһҳбһё)бҹ” бһңбһ¶бһ”бҹ’бһҡбҹҖбһ”бһ”бһ¶бһ“бһ‘бҹ…бһ“бһ№бһ„бһӮбһ»бһҖбһңбһ„бҹ’бһңбҹҒбһ„бһ•бҹ’бһӣбһјбһңбһҠбҹҸбһӣбҹ’бһ”бһёбһӣбҹ’бһ”бһ¶бһүбһ”бҹҶбһ•бһ»бһҸбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҡбһҝбһ„бһ–бҹ’бһҡбҹҒбһ„бһҖбҹ’бһҡбһ·бһҖбһ”бһ»бһҡбһ¶бһҺ бһҠбҹӮбһӣбһҳбһ¶бһ“бһҲбҹ’бһҳбҹ„бҹҮбһҗбһ¶ **The Labyrinth**бҹ”

бһӮбһ»бһҖбһңбһ„бҹ’бһңбҹҒбһ„бһ•бҹ’бһӣбһјбһңбһ“бҹҒбҹҮ бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбһ¶бһ„бһҹбһ„бҹӢбһЎбһҫбһ„бһҷбҹүбһ¶бһ„бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһ”бҹҶбһ•бһ»бһҸ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҒбһ¶бҹҶбһ„бһҹбһҸбҹ’бһңбһ…бһҳбҹ’бһӣбҹӮбһҖбһҹбҹҠбһёбһҹбһ¶бһ…бҹӢбһҳбһ“бһ»бһҹбҹ’бһҹбһҲбҹ’бһҳбҹ„бҹҮ **Minotaur (бһҳбһёбһҺбһјбһҸбҹҗбһҡ)**бҹ” бһўбҹ’бһ“бһҖбһҺбһ¶бһҠбҹӮбһӣбһҠбһҫбһҡбһ…бһјбһӣбһ‘бҹ…бһҒбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„ бһӮбһәбһӮбҹ’бһҳбһ¶бһ“бһҗбҹ’бһ„бҹғбһўбһ¶бһ…бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ•бҹ’бһӣбһјбһңбһ…бҹҒбһүбһҳбһҖбһҖбҹ’бһҡбҹ…бһңбһ·бһүбһ”бһ¶бһ“бһЎбһҫбһҷ бһ бһҫбһҷбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһң Minotaur бһҹбҹҠбһёбһҮбһ¶бһўбһ¶бһ бһ¶бһҡбҹ”

бһ“бҹ…бһ–бҹҒбһӣбһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҖбҹ’бһӣбһ¶бһҷбһҮбһ¶ Spaghetti Code бһ“бҹ„бҹҮбһӮбһҳбҹ’бһҡбҹ„бһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҖбҹ’бһӣбһ¶бһҷбһҮбһ¶ В«LabyrinthВ» бһҳбһҪбһҷ бһҠбҹӮбһӣбһҹбһјбһҳбҹ’бһ”бһёбһҸбҹӮбһңбһ·бһҹбҹ’бһңбһҖбһҡбһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһңбһ¶бһҠбҹҶбһ”бһјбһ„ бһҖбҹҸбһңбһ„бҹ’бһңбҹҒбһ„бһ•бҹ’бһӣбһјбһңбһҡбһҖбһ…бҹ’бһҡбһҖбһ…бҹҒбһүбһҳбһ·бһ“бһғбһҫбһүбһҠбҹӮбһҡбҹ”

## бҹЎ. бһ”бһүбҹ’бһ бһ¶бҹ– бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһ”бһ¶бһ“бһҮбһ¶бһҖбһјбһҠбһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһҖбҹ’бһӣбһ¶бһҷбһҮбһ¶бһӮбһ»бһҖбһңбһ„бҹ’бһңбҹҒбһ„бһ•бҹ’бһӣбһјбһң? (The Issue: The Labyrinth of Code)

Spaghetti Code бһ¬бһҖбһјбһҠбһҡбһүбҹүбҹҒбһҡбһүбҹүбҹғ бһҖбһҫбһҸбһЎбһҫбһ„бһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһңбһ·бһҹбҹ’бһңбһҖбһҡбһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠбһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢ (No Architecture)бҹ” бһ–бһҪбһҖбһӮбҹҒбһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ–бһёбһ•бҹ’бһ“бҹӮбһҖбһҳбһҪбһҷбһ‘бҹ…бһ•бҹ’бһ“бҹӮбһҖбһҳбһҪбһҷбһ‘бҹҖбһҸбһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢ бһ“бһ·бһ„бһӮбҹ’бһҳбһ¶бһ“бһҡбһ”бҹҖбһ”бһҡбҹҖбһ”бһҡбһҷбҹ”

бһҷбһјбһҡбҹ—бһ‘бҹ… бһ“бҹ…бһ–бҹҒбһӣбһӮбһҳбҹ’бһҡбҹ„бһ„бһҖбһ¶бһ“бҹӢбһҸбҹӮбһ’бҹҶбһЎбһҫбһ„бҹ— бһ“бһ·бһ„бһҳбһ¶бһ“бһҳбһ»бһҒбһ„бһ¶бһҡбһҖбһ¶бһ“бҹӢбһҸбҹӮбһ…бҹ’бһҡбһҫбһ“ бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҡбһңбһ¶бһ„бһҖбһјбһҠбһ“бһёбһҳбһҪбһҷбҹ—бһ“бһ№бһ„бһҖбһ¶бһ“бҹӢбһҸбҹӮбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһЎбһҫбһ„бҹ— бһҡбһ бһјбһҸбһҠбһӣбҹӢбһӮбҹ’бһҳбһ¶бһ“бһ“бһҡбһҺбһ¶бһҳбҹ’бһ“бһ¶бһҖбҹӢбһўбһ¶бһ…бһҳбһҫбһӣбһғбһҫбһүбһҮбһ¶бһҡбһјбһ”бһ—бһ¶бһ–бһ’бҹҶ (Big Picture) бһЎбһҫбһҷбҹ” 

бһӮбҹ’бһҡбҹ„бҹҮбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҠбҹҸбһ’бҹҶбһ”бҹҶбһ•бһ»бһҸбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Labyrinth бһӮбһәбһҖбһ¶бһҡбһӣбһ¶бһҖбҹӢбһҒбҹ’бһӣбһҪбһ“бһ“бҹғбһҹбһҸбҹ’бһңбһ…бһҳбҹ’бһӣбҹӮбһҖ Minotaur бһҠбҹӮбһӣбһҮбһ¶бһ“бһ·бһҳбһ·бһҸбҹ’бһҸбһҡбһјбһ”бһ“бҹғ **Hidden Bugs (бһҖбҹҶбһ бһ»бһҹбһӣбһ¶бһҖбҹӢбһҖбҹҶбһ”бһ¶бҹҶбһ„бһҠбҹҸбһ’бҹ’бһ„бһ“бҹӢбһ’бҹ’бһ„бһҡ)**бҹ” бһңбһ·бһҹбҹ’бһңбһҖбһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҳбһ¶бһ“бһ—бһ¶бһ–бһ—бҹҗбһҷбһҒбҹ’бһӣбһ¶бһ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҖбҹӮбһ”бҹ’бһҡбҹӮбһҖбһјбһҠбһ…бһ¶бһҹбҹӢбҹ— (Fear-driven Development) бһ–бҹ’бһҡбҹ„бҹҮбһҒбҹ’бһӣбһ¶бһ…бһ”бҹүбҹҮбһ…бҹҶ Minotaur бһ бһҫбһҷбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ‘бһ¶бҹҶбһ„бһҳбһјбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһҖбҹӢбһҹбҹ’бһҡбһ»бһҸбһ…бһ»бҹҮбҹ” бһ‘бһёбһ”бҹҶбһ•бһ»бһҸ бһҖбҹ’бһҡбһ»бһҳбһ бҹҠбһ»бһ“бһҸбҹ’бһҡбһјбһңбһ…бҹҶбһҺбһ¶бһҷбһҗбһңбһ·бһҖбһ¶бһҡбһ¶бһ”бҹӢбһҹбҹӮбһ“бһҠбһ»бһӣбҹ’бһӣбһ¶бһҡбһҮбһ¶бһҡбҹҖбһ„бһҡбһ¶бһӣбҹӢбһҒбҹӮ бһҠбһҫбһҳбҹ’бһ”бһёбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһҗбҹӮбһ‘бһ¶бҹҶбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҠбҹӮбһӣбһӣбҹӮбһ„бһ…бһ„бҹӢбһҠбһҫбһҡбһ‘бҹ…бһҳбһ»бһҒбһ“бҹҒбҹҮ (Technical Debt)бҹ”

## бҹў. бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бһҖбҹ’бһ“бһ»бһ„бһ–бһ·бһ—бһ–бһ–бһ·бһҸ

бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣ **бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бһ…бҹҶбһ“бһҪбһ“ бҹҘ** бһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһҖбһјбһҠбһҡбһүбҹүбҹҒбһҡбһүбҹүбҹғбһ”бҹҶбһ•бҹ’бһӣбһ¶бһүбһ•бһӣбһ·бһҸбһ—бһ¶бһ–бһҖбһ¶бһҡбһ„бһ¶бһҡ бһ“бһ·бһ„бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–

---

### бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһё бҹЎ вҖ” бһҖбһјбһҠбһҠбҹӮбһӣбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһӮбҹ’бһ“бһ¶бһҹбҹ’бһўбһ·бһҸбһҡбһҳбһҪбһҸ (Tightly Coupled Modules)

**бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бҹ–** бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ App бһӣбһҖбҹӢбһ‘бҹҶбһ“бһ·бһүбһҳбһҪбһҷ бһҳбһ»бһҒбһ„бһ¶бһҡбһӮбһҺбһ“бһ¶бһ”бҹ’бһҡбһ¶бһҖбҹӢбһ–бһ“бҹ’бһ’ (Tax Module) бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбһҡбһҹбҹҒбһҡбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢ (Hard-coupled) бһ‘бҹ…бһ“бһ№бһ„бһ‘бҹҶбһ–бҹҗбһҡбһҡбһ…бһ“бһ¶бһҖбһ¶бһҡбһ‘бһјбһ‘бһ¶бһҸбҹӢбһ”бҹ’бһҡбһ¶бһҖбҹӢ (Checkout UI Container)бҹ”

*   **бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһўбһҹбһҖбһҳбҹ’бһҳ / Low EQ / бһҖбҹҶбһ бһ»бһҹбһҶбҹ’бһӮбһ„бҹ–** бһңбһ·бһҹбҹ’бһңбһҖбһҡбһ…бһ„бҹӢбһҖбҹӮбһӣбһҳбҹ’бһўбһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһ‘бҹҶбһ–бҹҗбһҡ Checkout бһұбҹ’бһҷбһҹбҹ’бһўбһ¶бһҸбһҮбһ¶бһ„бһҳбһ»бһ“бҹ” бһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһ–бһҪбһҖбһӮбҹҒбһҖбҹӮбһ”бҹ’бһҡбҹӮ UI бһҹбҹ’бһҡбһ¶бһ”бҹӢбһҸбҹӮбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһӮбһҺбһ“бһ¶бһ”бҹ’бһҡбһ¶бһҖбҹӢбһ–бһ“бҹ’бһ’бһҸбҹ’бһҡбһјбһңбһӮбһ¶бҹҶбһ„бһҳбһ·бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһҠбҹ„бһҷбһҹбһ¶бһҡбһҸбҹӮбһҖбһјбһҠбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһ•бҹ’бһ“бҹӮбһҖбһ“бҹҒбҹҮбһҳбһ¶бһ“бһҖбһ¶бһҡбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бһӮбҹ’бһ“бһ¶бһҹбҹ’бһўбһ·бһҸбһҡбһҳбһҪбһҸбһҒбҹ’бһӣбһ¶бҹҶбһ„бһ–бҹҒбһҖ (Spaghetti dependency)бҹ” бһ–бһҪбһҖбһӮбҹҒбһҸбҹ’бһҡбһјбһңбһ…бҹҶбһҺбһ¶бһҷбһ–бҹҒбһӣ бҹЈ бһҗбҹ’бһ„бҹғ бһҠбһҫбһҳбҹ’бһ”бһёбһҸбһ¶бһҳбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ бһ“бһ·бһ„бһҮбһҪбһҹбһҮбһ»бһӣбһҖбһјбһҠбһЎбһҫбһ„бһңбһ·бһүбҹ”
*   **бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҹбҹ’бһҗбһ¶бһ”бһ“бһ¶ / High EQ / бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–** бһўбһ“бһ»бһңбһҸбҹ’бһҸ **Decoupling (Loose Coupling) & Interface Design**бҹ” бһҸбҹ’бһҡбһјбһңбһҡбһ…бһ“бһ¶бһұбҹ’бһҷбһҳбһ»бһҒбһ„бһ¶бһҡбһӮбһҺбһ“бһ¶бһ–бһ“бҹ’бһ’ бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбһ¶бһ…бҹӢбһҠбҹ„бһҷбһЎбҹӮбһҖбһ–бһё UI бһ‘бһ¶бҹҶбһ„бһҹбҹ’бһҡбһ»бһ„бһҸбһ¶бһҳбһҡбһҷбҹҲ API бһ¬ Interface бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбҹ” бһ‘бҹҶбһ–бҹҗбһҡ Checkout бһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡбһ–бһ“бҹ’бһ’бһҳбһҖбһ”бҹ’бһҡбһҫ бһҸбҹӮбһҳбһ·бһ“бһҒбҹ’бһңбһӣбҹӢбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһңбһ¶бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҒбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„бһЎбһҫбһҷбҹ”
*   **бһӣбһ‘бҹ’бһ’бһ•бһӣбҹ–** бһҖбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠбһҮбҹҶбһ–бһ¶бһҖбҹӢбһӮбҹ’бһ“бһ¶бһҠбһјбһ…бһҹбһҡбһҹбҹғбһҳбһёбһ“бһ¶бҹҶбһұбҹ’бһҷбһ”бҹүбҹҮбһҖбһ“бҹ’бһӣбҹӮбһ„бһҳбһҪбһҷ бһҒбһјбһ…бһҒбһ¶бһҸбһҖбһ“бҹ’бһӣбҹӮбһ„бһҳбһҪбһҷбһ‘бҹҖбһҸбһӮбҹ’бһҳбһ¶бһ“бһҲбһ”бҹӢбһҲбһҡбҹ” бһҖбһ¶бһҡбһ”бҹҶбһ”бҹӮбһҖбһҖбһјбһҠбһұбҹ’бһҷбһҳбһ¶бһ“бһҜбһҖбһҡбһ¶бһҮбҹ’бһҷбһ—бһ¶бһ–бһҮбһҪбһҷбһұбҹ’бһҷбһҖбһ¶бһҡбһҖбҹӮбһӣбһҳбҹ’бһўбһҖбһ¶бһҡбһҡбһ…бһ“бһ¶ UI бһҳбһ¶бһ“бһ—бһ¶бһ–бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣ бһӣбһҝбһ“бһҡбһ бҹҗбһҹ бһ“бһ·бһ„бһӮбҹ’бһҳбһ¶бһ“бһ•бһӣбһ”бҹүбҹҮбһ–бһ¶бһӣбҹӢбһҠбһӣбҹӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҹбҹ’бһ“бһјбһӣбҹ”

### бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһё бҹў вҖ” бһҖбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠбһҠбҹ„бһҷбһ—бһ¶бһ–бһ—бҹҗбһҷбһҒбҹ’бһӣбһ¶бһ… (Fear-Driven Development & Dead Code)

**бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бҹ–** бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ…бһ¶бһҹбҹӢбһҳбһҪбһҷ бһҳбһ¶бһ“бһҜбһҖбһҹбһ¶бһҡбһҖбһјбһҠ (Function) бһ…бҹҶбһ“бһҪбһ“ бҹўбҹ  бһҠбҹӮбһӣбһҳбһҫбһӣбһ‘бҹ…бһ бһ¶бһҖбҹӢбһҠбһјбһ…бһҮбһ¶бһӣбҹӮбһ„бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҡбһҷбҹҲбһ–бҹҒбһӣ бҹЈ бһҶбҹ’бһ“бһ¶бҹҶбһҳбһҖбһ бһҫбһҷбҹ”

*   **бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһўбһҹбһҖбһҳбҹ’бһҳ / Low EQ / бһҖбҹҶбһ бһ»бһҹбһҶбҹ’бһӮбһ„бҹ–** бһңбһ·бһҹбҹ’бһңбһҖбһҡбһҗбҹ’бһҳбһёбһ…бһ„бҹӢбһӣбһ»бһ”бһҖбһјбһҠбһ…бһ¶бһҹбҹӢбҹ—бһ‘бһ¶бҹҶбһ„бһ“бҹ„бҹҮбһ…бҹ„бһӣ бһҠбһҫбһҳбҹ’бһ”бһёбһұбҹ’бһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҹбҹ’бһҡбһ¶бһӣ бһ“бһ·бһ„бһҹбҹ’бһўбһ¶бһҸбҹ” бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮ бһңбһ·бһҹбҹ’бһңбһҖбһҡбһҮбһ¶бһ“бҹӢбһҒбҹ’бһ–бһҹбҹӢбһ”бһ¶бһ“бһҹбҹ’бһҡбҹӮбһҖбһғбһ¶бһҸбҹӢбһ—бҹ’бһӣбһ¶бһҳбҹ—бһҗбһ¶бҹ– *В«бһҖбһ»бҹҶбһӣбһ»бһ”бһұбҹ’бһҷбһҹбҹ„бҹҮ! бһ‘бҹ„бҹҮбһ”бһёбһҮбһ¶бһҷбһҫбһ„бһҳбһ·бһ“бһҠбһ№бһ„бһҗбһ¶бһңбһ¶бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ’бҹ’бһңбһҫбһўбҹ’бһңбһёбһҖбҹҸбһҠбҹ„бһҷ бһҸбҹӮбһ”бһҫбһӣбһ»бһ”бһ…бҹ„бһӣ бһҒбҹ’бһӣбһ¶бһ…бһҖбҹ’бһҡбҹӮбһ„бһңбһ¶бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ‘бһ¶бҹҶбһ„бһҳбһјбһӣбһҠбһҪбһӣбһҡбһӣбҹҶ бһ‘бһ»бһҖбһңбһ¶бһ…бҹ„бһӣбһ”бҹӮбһ”бһ бҹ’бһ“бһ№бһ„бһ‘бҹ…!В»* бһ“бҹҒбҹҮбһҮбһ¶бһҖбһ¶бһҡбһҡбһҹбҹӢбһ“бҹ…бһҮбһ¶бһҳбһҪбһҷ Minotaur бһҖбҹ’бһ“бһ»бһ„ Labyrinth бһҠбҹ„бһҷбһҹбһ¶бһҡбһӮбҹ’бһҳбһ¶бһ“ Unit Tests бһҖбһ¶бһҡбһ–бһ¶бһҡбҹ”

### бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһё бҹЈ вҖ” бһҖбһ¶бһҡбһҖбҹӮбһ”бҹ’бһҡбҹӮбһҖбһјбһҠбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһӣбһҫ Server бһ–бһ·бһҸ (No Version Control & Direct FTP Editing)

**бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бҹ–** бһҖбҹ’бһҡбһ»бһҳбһ бҹҠбһ»бһ“бһҳбһҪбһҷбһҳбһ¶бһ“бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһӣбһҖбҹӢбһ‘бҹҶбһ“бһ·бһүбһҳбһҪбһҷбҹ” бһҗбҹ’бһ„бҹғбһҳбһҪбһҷ бһҳбһ¶бһ“бһҖбҹҶбһ бһ»бһҹбһҸбһјбһ…бһҳбһҪбһҷбһҖбһҫбһҸбһЎбһҫбһ„бһӣбһҫбһ‘бҹҶбһ–бҹҗбһҡбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһҡбһ”бһҹбҹӢбһўбһҸбһ·бһҗбһ·бһҮбһ“бҹ”

*   **бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһўбһҹбһҖбһҳбҹ’бһҳ / Low EQ / бһҖбҹҶбһ бһ»бһҹбһҶбҹ’бһӮбһ„бҹ–** бһңбһ·бһҹбҹ’бһңбһҖбһҡбһҳбҹ’бһ“бһ¶бһҖбҹӢбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ FTP бһ…бһјбһӣбһ‘бҹ…бһ”бһҫбһҖбһҜбһҖбһҹбһ¶бһҡбһҖбһјбһҠ бһҡбһҪбһ…бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҖбҹӮбһ”бҹ’бһҡбҹӮбһҖбһјбһҠбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһ“бҹ…бһӣбһҫ Server бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһ–бһ·бһҸ (Production Server) бһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ Git бһ¬бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ Version Control бһЎбһҫбһҷ бһ–бҹ’бһҡбҹ„бҹҮбһӮбһ·бһҸбһҗбһ¶бһҖбҹӮбһҸбҹӮбһҳбһҪбһҷбһ—бҹ’бһӣбҹӮбһҸбһҡбһҪбһ…бһ бһҫбһҷбҹ” бһӮбһ¶бһҸбҹӢбһ”бһ¶бһ“бһңбһ¶бһҷбһўбһҖбҹ’бһҹбһҡбһҒбһ»бһҹбһғбҹ’бһӣбһ¶ (Syntax Error) бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһ‘бһ¶бҹҶбһ„бһҳбһјбһӣбһҸбҹ’бһҡбһјбһңбһӮбһ¶бҹҶбһ„бһҠбһҪбһӣбһҡбһӣбҹҶ (White Screen of Death) бһ бһҫбһҷбһӮбһ¶бһҸбҹӢбһҳбһ·бһ“бһҠбһ№бһ„бһҸбҹ’бһҡбһјбһң Revert бһҸбҹ’бһҡбһЎбһ”бҹӢбһҳбһҖбһҖбһјбһҠбһ…бһ¶бһҹбҹӢбһңбһ·бһүбһҠбҹ„бһҷбһҡбһ”бҹҖбһ”бһҺбһ¶бһЎбһҫбһҷбҹ”
*   **бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҹбҹ’бһҗбһ¶бһ”бһ“бһ¶ / High EQ / бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–** бһ”бһ„бҹ’бһ бһ¶бһҸбҹӢбһҖбҹ’бһҡбһ»бһҳбһҖбһ¶бһҡбһ„бһ¶бһҡбһұбҹ’бһҷбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ **Git Version Control & CI/CD Pipeline** бһҷбҹүбһ¶бһ„бһҸбһ№бһ„бһҡбҹүбһ№бһ„бһ”бҹҶбһ•бһ»бһҸбҹ” бһӮбҹ’бһҳбһ¶бһ“бһ“бһҡбһҺбһ¶бһҳбҹ’бһ“бһ¶бһҖбҹӢбһҳбһ¶бһ“бһҹбһ·бһ‘бҹ’бһ’бһ·бһҖбҹӮбһ”бҹ’бһҡбҹӮбһҖбһјбһҠбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһӣбһҫ Server бһ–бһ·бһҸбһЎбһҫбһҷбҹ” бһҡбһ¶бһӣбҹӢбһҖбһ¶бһҡбһҖбҹӮбһ”бҹ’бһҡбҹӮбһҸбҹ’бһҡбһјбһңбһ’бҹ’бһңбһҫбһ“бҹ…бһӣбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“ (Local) бһҡбһҪбһ…бһҡбһ»бһүбһ‘бҹ…бһҖбһ¶бһ“бҹӢ Git Branch бһ“бһ·бһ„бһҶбҹ’бһӣбһ„бһҖбһ¶бһҸбҹӢбһҖбһ¶бһҡбһҸбҹҒбһҹбҹ’бһҸбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһҳбһ»бһ“бһ“бһ№бһ„бһҠбҹҶбһЎбһҫбһ„бһ‘бҹ… Serverбҹ”
*   **бһӣбһ‘бҹ’бһ’бһ•бһӣбҹ–** бһҖбһ¶бһҡбһҖбҹӮбһҖбһјбһҠбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһӣбһҫ Server бһ“бһ¶бҹҶбһұбҹ’бһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ„бһ¶бһҷбһҡбһ„бһӮбҹ’бһҡбҹ„бҹҮбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҠбһҪбһӣбһҡбһӣбҹҶбһ—бҹ’бһӣбһ¶бһҳбҹ— бһ“бһ·бһ„бһ–бһ·бһ”бһ¶бһҖбһҹбҹ’бһҸбһ¶бһҡбһЎбһҫбһ„бһңбһ·бһүбҹ” бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ Git бһ“бһ·бһ„ CI/CD бһҮбһҪбһҷбһ’бһ¶бһ“бһ¶бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҖбһјбһҠ бһ“бһ·бһ„бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһҹбҹ’бһҸбһ¶бһҡбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһЎбһҫбһ„бһңбһ·бһүбһҸбҹ’бһҡбһ№бһҳ бҹЎ бһңбһ·бһ“бһ¶бһ‘бһёбһ”бһҫбһҳбһ¶бһ“бһ”бһүбҹ’бһ бһ¶бҹ”

### бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһё бҹӨ вҖ” бһҜбһҖбһҹбһ¶бһҡбһҖбһјбһҠбһҷбһҖбҹ’бһҹбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҡбһ¶бһӣбҹӢбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһ„бһ¶бһҡ (Huge God Classes)

**бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бҹ–** бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһӮбһҳбҹ’бһҡбҹ„бһ„бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ App бһҳбһҪбһҷ бһҜбһҖбһҹбһ¶бһҡбһҲбҹ’бһҳбҹ„бҹҮ `UserController.js` бһҳбһ¶бһ“бһ”бҹ’бһҡбһңбҹӮбһ„бһҠбһӣбҹӢбһ‘бҹ… бҹЎбҹ ,бҹ бҹ бҹ  бһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбҹ” бһңбһ¶бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҸбһ¶бҹҶбһ„бһ–бһё бһҖбһ¶бһҡбһ…бһ»бҹҮбһҲбҹ’бһҳбҹ„бҹҮбһ”бһ»бһӮбҹ’бһӮбһӣбһ·бһҖ, бһҖбһ¶бһҡбһ•бҹ’бһүбһҫбһўбҹҠбһёбһҳбҹӮбһӣ, бһҖбһ¶бһҡбһӮбһҺбһ“бһ¶бһ”бҹ’бһҡбһ¶бһҖбҹӢбһҒбҹӮ, бһҖбһ¶бһҡбһ‘бһјбһ‘бһ¶бһҸбҹӢбһӣбһ»бһҷ бһҡбһ бһјбһҸбһҠбһӣбҹӢбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһ”бһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбҹ”

*   **бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһўбһҹбһҖбһҳбҹ’бһҳ / Low EQ / бһҖбҹҶбһ бһ»бһҹбһҶбҹ’бһӮбһ„бҹ–** бһҡбһ¶бһӣбҹӢбһ–бҹҒбһӣбһ…бһ„бҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳбһҳбһ»бһҒбһ„бһ¶бһҡбһҗбҹ’бһҳбһёбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһ“бһ№бһ„бһ”бһ»бһӮбҹ’бһӮбһӣбһ·бһҖ бһңбһ·бһҹбҹ’бһңбһҖбһҡбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһҹбһҡбһҹбҹӮбһҡбһҖбһјбһҠбһ”бһ“бҹ’бһҗбҹӮбһҳбһ…бһјбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡбһҷбһҖбҹ’бһҹбһ“бҹ„бҹҮбһҮбһ¶бһҡбҹҖбһ„бһҡбһ¶бһӣбҹӢбһҗбҹ’бһ„бҹғ бһ–бҹ’бһҡбҹ„бҹҮбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҳбһ·бһ“бһ”бһ¶бһ…бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһҜбһҖбһҹбһ¶бһҡбһҗбҹ’бһҳбһёбҹ” бһҷбһјбһҡбҹ—бһ‘бҹ… бһӮбҹ’бһҳбһ¶бһ“бһ“бһҡбһҺбһ¶бһҳбҹ’бһ“бһ¶бһҖбҹӢбһ бҹҠбһ¶бһ“бһ”бһҫбһҖбһҜбһҖбһҹбһ¶бһҡбһ“бҹ„бҹҮбһЎбһҫбһҷ бһ–бҹ’бһҡбҹ„бҹҮбһңбһ¶бһ’бҹҶбһ–бҹҒбһҖ бһҠбһҫбһҡбһҷбһәбһҸ бһ“бһ·бһ„бһ–бҹ„бһҡбһ–бҹҒбһүбһҠбҹ„бһҷбһҖбһјбһҠбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһҡбһҖбһ•бҹ’бһӣбһјбһңбһҳбһ·бһ“бһғбһҫбһү (A Labyrinth within a file)бҹ”
*   **бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҹбҹ’бһҗбһ¶бһ”бһ“бһ¶ / High EQ / бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–** бһўбһ“бһ»бһңбһҸбҹ’бһҸбһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚ **Single Responsibility Principle (SRP)**бҹ” бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ”бҹҶбһ”бҹӮбһҖбһҜбһҖбһҹбһ¶бһҡбһҷбһҖбҹ’бһҹбһ“бҹ„бҹҮбһҮбһ¶бһҜбһҖбһҹбһ¶бһҡбһҸбһјбһ…бҹ—бһҠбҹӮбһӣбһҳбһ¶бһ“бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҸбҹӮбһҳбһҪбһҷбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢ (бһҠбһјбһ…бһҮбһ¶ `RegistrationService`, `EmailNotifier`, `PayrollCalculator`) бһҠбһҫбһҳбҹ’бһ”бһёбһұбҹ’бһҷбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҷбһӣбҹӢ бһ“бһ·бһ„бһҸбҹҒбһҹбҹ’бһҸбҹ”
*   **бһӣбһ‘бҹ’бһ’бһ•бһӣбҹ–** бһҖбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠбһҡбһ¶бһ”бҹӢбһҳбҹүбһәбһ“бһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡбһҸбҹӮбһҳбһҪбһҷбһ“бһ¶бҹҶбһұбҹ’бһҷбһҖбһјбһҠбһ–бһ·бһ”бһ¶бһҖбһўбһ¶бһ“ бһ–бһ·бһ”бһ¶бһҖбһҷбһӣбҹӢ бһ“бһ·бһ„бһ„бһ¶бһҷбһ”бһ„бҹ’бһҖбһҖбҹҶбһ бһ»бһҹбһҮбһ¶бһ“бҹӢбһӮбҹ’бһ“бһ¶бҹ” бһҖбһ¶бһҡбһ”бҹҶбһ”бҹӮбһҖбһҖбһјбһҠбһҸбһ¶бһҳбһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚ SRP бһҮбһҪбһҷбһұбҹ’бһҷбһҖбһјбһҠбһҳбһ¶бһ“бһҡбһ”бҹҖбһ”бһҡбҹҖбһ”бһҡбһҷ бһҹбҹ’бһўбһ¶бһҸ бһ“бһ·бһ„бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһ”бһ“бҹ’бһҸбҹ”

### бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһё бҹҘ вҖ” бһҖбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһҸбһҳбҹ’бһӣбҹғбһҖбһјбһҠбһҡбһ№бһ„бһӮбҹ’бһҳбһ¶бһ“бһӣбҹҶбһ“бһ¶бҹҶ (Magic Numbers & Hardcoded Configurations)

**бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бҹ–** бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһјбһҠбһӮбһҳбҹ’бһҡбҹ„бһ„бһ‘бһјбһ‘бһ¶бһҸбҹӢбһ”бҹ’бһҡбһ¶бһҖбҹӢ бһңбһ·бһҹбҹ’бһңбһҖбһҡбһ”бһ¶бһ“бһҹбһҡбһҹбҹҒбһҡбһҸбһҳбҹ’бһӣбҹғбһҡбһ№бһ„ (Hardcoded) бһҠбһјбһ…бһҮбһ¶ IP Address бһҡбһ”бһҹбҹӢ Server бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ бһ“бһ·бһ„ API Keys бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһјбһҠбһҡбһ¶бһ”бҹӢбһҡбһҷбһҖбһ“бҹ’бһӣбҹӮбһ„бҹ”

*   **бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһўбһҹбһҖбһҳбҹ’бһҳ / Low EQ / бһҖбҹҶбһ бһ»бһҹбһҶбҹ’бһӮбһ„бҹ–** бһ“бҹ…бһ–бҹҒбһӣбһҖбҹ’бһҡбһ»бһҳбһ бҹҠбһ»бһ“бһ”бҹ’бһҸбһјбһҡбһҳбҹүбһ¶бһҹбҹҠбһёбһ“ Server бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһўбһ¶бһҹбһҷбһҠбҹ’бһӢбһ¶бһ“ IP бһҗбҹ’бһҳбһё бһҖбҹ’бһҡбһ»бһҳбһңбһ·бһҹбҹ’бһңбһҖбһҡбһҸбҹ’бһҡбһјбһңбһ…бҹҶбһҺбһ¶бһҷбһ–бҹҒбһӣбһ–бҹҒбһүбһҳбһҪбһҷбһҷбһ”бҹӢ бһўбһ„бҹ’бһӮбһ»бһҷбһ”бһҫбһҖбһҜбһҖбһҹбһ¶бһҡбһҖбһјбһҠбһҡбһ¶бһ”бҹӢбһҡбһҷ бһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹӮбһ”бҹ’бһҡбҹӮбһӣбҹҒбһҒ IP бһ“бҹ„бҹҮбһҳбҹ’бһҸбһ„бһҳбһҪбһҷбҹ—бҹ” бһ–бһҪбһҖбһӮбҹҒбһ”бһ¶бһ“бһ—бҹ’бһӣбҹҒбһ…бһҖбҹӮбһ”бҹ’бһҡбҹӮбһҖбһјбһҠбһ“бҹ…бһҖбһ“бҹ’бһӣбҹӮбһ„бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“ бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһўбһҸбһ·бһҗбһ·бһҮбһ“бһҳбһ·бһ“бһўбһ¶бһ…бһ‘бһјбһ‘бһ¶бһҸбҹӢбһ”бҹ’бһҡбһ¶бһҖбҹӢбһ”бһ¶бһ“бһ“бҹ…бһ–бҹ’бһҡбһ№бһҖбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбҹ”
*   **бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҹбҹ’бһҗбһ¶бһ”бһ“бһ¶ / High EQ / бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–** бһўбһ“бһ»бһңбһҸбҹ’бһҸ **Centralized Environment Variables (`.env`)**бҹ” бһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһҹбһҡбһҹбҹҒбһҡбһҸбһҳбҹ’бһӣбҹғбһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ бһ¬бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҡбһ№бһ„бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһјбһҠбһЎбһҫбһҷбҹ” бһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһҳбһјбһӣбһ•бҹ’бһҸбһ»бҹҶбһңбһ¶бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҸбҹӮбһҳбһҪбһҷ (`config.js` бһ¬ `.env`) бһҠбһҫбһҳбҹ’бһ”бһёбһұбҹ’бһҷбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҖбҹӮбһ”бҹ’бһҡбҹӮбһҸбҹӮбһҳбһҪбһҷбһҖбһ“бҹ’бһӣбҹӮбһ„ бһ“бһ·бһ„бһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢбҹ”
*   **бһӣбһ‘бҹ’бһ’бһ•бһӣбҹ–** бһҖбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһҸбһҳбҹ’бһӣбҹғбһҡбһ№бһ„бһҖбҹ’бһ“бһ»бһ„бһҖбһјбһҠбһ“бһ¶бҹҶбһұбҹ’бһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ–бһ·бһ”бһ¶бһҖбһ”бһҸбҹӢбһ”бҹӮбһ“ бһ“бһ·бһ„бһ„бһ¶бһҷбһ’бҹ’бһӣбһ¶бһҷбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ…бҹ” бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҖбһҺбҹ’бһҸбһ¶бһӣбһҮбһҪбһҷбһұбҹ’бһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бһҸбҹӢбһ”бҹӮбһ“бһӣбһҝбһ“ бһ“бһ·бһ„бһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢбһ”бҹҶбһ•бһ»бһҸбҹ”

## бҹЈ. бһҖбһҸбҹ’бһҸбһ¶бһҮбһҳбҹ’бһҡбһ»бһүбҹ– бһ—бһ¶бһ–бһ”бҹ’бһҡбһүбһ¶бһ”бҹӢбһ”бҹ’бһҡбһүбһ¶бһӣбҹӢ бһ“бһ·бһ„бһҖбһ„бҹ’бһңбҹҮбһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ (The Aggravator: Rush Delivery & Lack of Architecture)

бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһ”бһ¶бһ“бһҮбһ¶бһҖбһјбһҠбһҠбҹҸбһҹбҹ’бһўбһ¶бһҸбһҹбҹ’бһўбҹҶбһҡбһ”бһҹбҹӢбһҷбһҫбһ„ бһ„бһ¶бһҷбһ“бһ№бһ„бһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣбһ‘бҹ…бһҮбһ¶ Spaghetti Code бһҠбҹҸбһӮбһҪбһҡбһұбҹ’бһҷбһҒбҹ’бһӣбһ¶бһ…бһҒбҹ’бһӣбһ¶бҹҶбһ„бһҳбҹ’бһӣбҹүбҹҒбҹҮ? бһҖбһҸбҹ’бһҸбһ¶бһҮбһҳбҹ’бһҡбһ»бһүбһҡбһҪбһҳбһҳбһ¶бһ“бҹ–

1.  **бһҹбһҳбҹ’бһ–бһ¶бһ’бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ”бҹ’бһҡбһүбһ¶бһ”бҹӢбһ”бҹ’бһҡбһүбһ¶бһӣбҹӢ (Feature Rush)бҹ–** бһ”бҹ’бһҡбһ’бһ¶бһ“бһ•бҹ’бһ“бҹӮбһҖбһўбһ¶бһҮбһёбһңбһҖбһҳбҹ’бһҳбһҸбҹӮбһ„бһҸбҹӮбһ…бһ„бҹӢбһ”бһ¶бһ“бһҳбһ»бһҒбһ„бһ¶бһҡбһҗбҹ’бһҳбһёбһұбҹ’бһҷбһ”бһ¶бһ“бһӣбһҝбһ“бһ”бҹҶбһ•бһ»бһҸбҹ” бһ–бһҪбһҖбһӮбҹҒбһҹбһ„бҹ’бһҖбһҸбҹӢбһҖбҹ’бһ”бһ¶бһӣбһңбһ·бһҹбҹ’бһңбһҖбһҡбһұбҹ’бһҷбһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠбһҡбһ»бһүбһ‘бҹ…бһҳбһ»бһҒбһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҡбһ…бһ“бһ¶бҹ– *В«бһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠбһұбҹ’бһҷбһҠбһҫбһҡбһ‘бҹ…бһҡбһҪбһ…бһҹбһ·бһ“бһ‘бҹ… бһҡбһҝбһ„бһҹбҹ’бһўбһ¶бһҸбһ…бһ¶бҹҶбһ’бҹ’бһңбһҫ Refactoring бһ–бҹҒбһӣбһҖбҹ’бһҡбҹ„бһҷ!В»* (бһҠбҹӮбһӣбһҖбһ¶бһҡбһ–бһ·бһҸ бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҖбҹ’бһҡбҹ„бһҷбһ“бҹ„бҹҮбһҳбһ·бһ“бһҠбҹӮбһӣбһҳбһҖбһҠбһӣбҹӢбһЎбһҫбһҷ)бҹ”
2.  **бһҖбһ„бҹ’бһңбҹҮбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҸбҹҒбһҹбҹ’бһҸбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸ (No Testing Culture)бҹ–** бһ“бҹ…бһ–бҹҒбһӣбһӮбҹ’бһҳбһ¶бһ“ Unit Tests бһңбһ·бһҹбҹ’бһңбһҖбһҡбһӮбҹ’бһҳбһ¶бһ“бһҒбҹӮбһӣбһҖбһ¶бһҡбһ–бһ¶бһҡбһЎбһҫбһҷбҹ” бһ–бһҪбһҖбһӮбҹҒбһҳбһ·бһ“бһ бҹҠбһ¶бһ“бһҖбҹӮбһ”бҹ’бһҡбҹӮбһҖбһјбһҠбһұбҹ’бһҷбһҹбҹ’бһўбһ¶бһҸ (Refactoring) бһ–бҹ’бһҡбҹ„бҹҮбһҒбҹ’бһӣбһ¶бһ…бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһҒбһјбһ…бһҒбһ¶бһҸбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ бһ‘бһҫбһ”бһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸбһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠбһҡбһүбҹүбҹҒбһҡбһүбҹүбҹғбһ”бһ“бҹ’бһҗбҹӮбһҳбһ–бһёбһӣбһҫбһҡбһ”бһҹбҹӢбһ…бһ¶бһҹбҹӢбһҮбһ¶бһҡбҹҖбһ„бһҡбһ¶бһӣбҹӢбһҗбҹ’бһ„бҹғбҹ”
3.  **бһҖбһ„бҹ’бһңбҹҮбһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ (Lack of Architectural Oversight)бҹ–** бһҖбһ¶бһҡбһҒбҹ’бһңбҹҮбһҸбһҪбһ“бһ¶бһ‘бһёбһҡбһ”бһҹбҹӢ Software Architect бһ¬ Senior Code Reviews бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһңбһ·бһҹбҹ’бһңбһҖбһҡбһҳбҹ’бһ“бһ¶бһҖбҹӢбҹ—бһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠбһҸбһ¶бһҳбһҸбҹӮбһ…бһ·бһҸбҹ’бһҸбһҒбҹ’бһӣбһҪбһ“бһҜбһ„ бһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһ”бһ‘бһҠбҹ’бһӢбһ¶бһ“бһҡбһҪбһҳбҹ”

## бҹӨ. бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ‘бһјбһ‘бҹ…бҹ– бһҒбҹ’бһҹбҹӮбһўбҹҶбһ”бҹ„бҹҮбһўбһ¶бһҡбһёбһҷбҹүбһ¶бһҠбһ“бһёбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһңбһ·бһҹбҹ’бһңбһҖбһҡ (The General Solution: Ariadne's Thread for Developers)

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҡбһҝбһ„бһ–бҹ’бһҡбҹҒбһ„бһҖбҹ’бһҡбһ·бһҖ бһңбһёбһҡбһ”бһ»бһҡбһҹ Theseus бһўбһ¶бһ…бһҡбһҹбҹӢбһҡбһ¶бһ“бһҳбһ¶бһ“бһҮбһёбһңбһ·бһҸ бһ“бһ·бһ„бһҹбһҳбҹ’бһӣбһ¶бһ”бҹӢ Minotaur бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Labyrinth бһ”бһ¶бһ“ бһҠбҹ„бһҷбһҹбһ¶бһҡбһҸбҹӮбһӮбһ¶бһҸбҹӢбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ **В«бһҹбһҡбһҹбҹғбһўбҹҶбһ”бҹ„бҹҮ (Ariadne's Thread)В»** бһҠбҹӮбһӣбһўбһ¶бһҡбһёбһҷбҹүбһ¶бһҠбһ“бһёбһ”бһ¶бһ“бһ•бҹ’бһҸбһӣбҹӢбһұбҹ’бһҷбҹ” бһӮбһ¶бһҸбҹӢбһ…бһ„бһҒбҹ’бһҹбҹӮбһўбҹҶбһ”бҹ„бҹҮбһ“бҹ„бҹҮбһ–бһёбһҳбһ¶бһҸбҹӢбһ‘бҹ’бһңбһ¶бһҡбһ…бһјбһӣ бһҡбһҪбһ…бһҠбһҫбһҡбһ…бһјбһӣбһ‘бҹ…бһҒбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„бҹ” бһ–бҹҒбһӣбһҹбһҳбҹ’бһӣбһ¶бһ”бҹӢ Minotaur бһҡбһҪбһ… бһӮбһ¶бһҸбҹӢбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһҠбһҫбһҡбһҸбҹ’бһҡбһЎбһ”бҹӢбһҗбһҷбһҖбҹ’бһҡбҹ„бһҷбһҸбһ¶бһҳбһҒбҹ’бһҹбҹӮбһўбҹҶбһ”бҹ„бҹҮбһ“бҹ„бҹҮбһҳбһҖбһҡбһҖбһ…бҹ’бһҡбһҖбһ…бҹҒбһүбһңбһ·бһүбһҠбҹ„бһҷбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бҹ”

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһңбһ·бһҹбҹҗбһҷ Software Engineering бһҷбһҫбһ„бһҖбҹҸбһҳбһ¶бһ“ **В«Ariadne's ThreadВ»** бһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһҷбһҫбһ„бһҠбҹӮбһҡбҹ–

1.  **бһҹбһҡбһҹбҹҒбһҡ Unit Tests бһҳбһ»бһ“бһ“бһ№бһ„бһҖбҹӮбһ”бҹ’бһҡбҹӮбһҖбһјбһҠ (Tests as Your Thread)бҹ–** бһҳбһ»бһ“бһ“бһ№бһ„бһҠбһҫбһҡбһ…бһјбһӣбһ‘бҹ…бһҖбҹӮбһ”бҹ’бһҡбҹӮбһҖбһјбһҠбһ…бһ¶бһҹбҹӢбҹ— бһҸбҹ’бһҡбһјбһңбһҹбһҡбһҹбҹҒбһҡ Unit Tests бһ–бҹҗбһ‘бҹ’бһ’бһҮбһ»бҹҶбһңбһ·бһүбһңбһ¶бһҮбһ¶бһҳбһ»бһ“бһҹбһ·бһ“бҹ” бһңбһ¶бһ”бҹ’бһҡбҹҖбһ”бһҠбһјбһ…бһҮбһ¶бһҖбһ¶бһҡбһ…бһ„бһҹбһҡбһҹбҹғбһўбҹҶбһ”бҹ„бҹҮбһўбҹҠбһёбһ…бһ№бһ„бҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҖбҹӮбһҖбһјбһҠбһҒбһ»бһҹ бһ“бҹ„бҹҮ Test бһ“бһ№бһ„бһӣбҹ„бһҸбһ–бһҺбҹҢбһҖбҹ’бһҡбһ бһҳ (Alert) бһ”бҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһұбҹ’бһҷбһҸбҹ’бһҡбһЎбһ”бҹӢбһҗбһҷбһҖбҹ’бһҡбҹ„бһҷ (Undo) бһҸбһ¶бһҳбһҒбҹ’бһҹбҹӮбһўбҹҶбһ”бҹ„бҹҮбһҳбһҖбһңбһ·бһүбһ—бҹ’бһӣбһ¶бһҳбҹ— бһҳбһ»бһ“бһ–бҹҒбһӣбһңбһ¶бһ’бҹ’бһӣбһ¶бһҷбһ‘бҹ…бһҠбһӣбҹӢбһҠбҹғбһўбһҸбһ·бһҗбһ·бһҮбһ“бҹ”
2.  **бһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶбһҖбһјбһҠбһұбҹ’бһҷбһҹбҹ’бһўбһ¶бһҸбһҮбһ¶бһ”бҹ’бһҡбһ…бһ¶бҹҶ (Regular Refactoring & Boy Scout Rule)бҹ–** бһ”бҹ’бһҡбһҖбһ¶бһ“бҹӢбһҒбҹ’бһҮбһ¶бһ”бҹӢбһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚ Boy Scoutбҹ– *В«бһ…бһјбһҡбһ‘бһ»бһҖбһұбҹ’бһҷбһҖбһ“бҹ’бһӣбҹӮбһ„бһ”бҹ„бҹҮбһҮбҹҶбһҡбҹҶ бһҹбҹ’бһўбһ¶бһҸбһҮбһ¶бһ„бһҳбһ»бһ“бһ–бҹҒбһӣбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҳбһҖбһҠбһӣбҹӢбһҮбһ¶бһ“бһ·бһ…бҹ’бһ…бҹ”В»* бһҡбһ¶бһӣбҹӢбһ–бҹҒбһӣбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһјбһӣбһ‘бҹ…бһҖбҹӮбһҖбһјбһҠ бһ…бһјбһҡбһҶбҹ’бһӣбҹҖбһҸбһ–бҹҒбһӣбһҮбһҳбҹ’бһҡбҹҮбһҖбһјбһҠбһҡбһүбҹүбҹҒбһҡбһүбҹүбҹғбһҸбһјбһ…бһҳбһҪбһҷбһ…бҹ„бһӣ бһҠбһҫбһҳбҹ’бһ”бһёбһҖбһ»бҹҶбһұбҹ’бһҷбһӮбһ»бһҖбһңбһ„бҹ’бһңбҹҒбһ„бһ•бҹ’бһӣбһјбһңбһ“бҹ„бҹҮбһ’бҹҶбһЎбһҫбһ„бҹ”
3.  **бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбҹҶбһҺбҹӮбһҖбһјбһҠбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢ (Strict Git Workflow)бҹ–** бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ Git Commits бһұбҹ’бһҷбһ”бһ¶бһ“бһӣбҹ’бһў бһ“бһ·бһ„бһүбһ№бһҖбһүбһ¶бһ”бҹӢбҹ” Commit бһ“бһёбһҳбһҪбһҷбҹ— бһӮбһәбһҮбһ¶бһ…бҹҶбһҺбһ»бһ…бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһ“бҹ…бһӣбһҫбһҒбҹ’бһҹбҹӮбһўбҹҶбһ”бҹ„бҹҮбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһҡбһҸбҹӢбһҗбһҷбһҖбҹ’бһҡбҹ„бһҷ (Revert) бһ”бһ¶бһ“бһҮбһ¶бһ“бһ·бһ…бҹ’бһ…бһ“бҹ…бһ–бҹҒбһӣбһңбһ„бҹ’бһңбҹҒбһ„бһ•бҹ’бһӣбһјбһңбҹ”

---

## бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҹбһ“бҹ’бһ“бһ·бһҠбҹ’бһӢбһ¶бһ“ (Conclusion)

**бһӮбһ»бһҖбһңбһ„бҹ’бһңбҹҒбһ„бһ•бҹ’бһӣбһјбһң бһ“бһ·бһ„бһҖбһјбһҠбһҡбһүбҹүбҹҒбһҡбһүбҹүбҹғ (Spaghetti Code)** бһӮбһәбһҮбһ¶бһҮбҹҶбһ„бһәбһҡбһ¶бһҸбһҸбҹ’бһ”бһ¶бһҸбһҠбҹҸбһ’бҹҶбһ”бҹҶбһ•бһ»бһҸбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһңбһ·бһҹбҹҗбһҷ Softwareбҹ” бһңбһ¶бһҳбһ·бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹӮбһ”бҹҶбһ•бҹ’бһӣбһ¶бһүбһ•бһӣбһ·бһҸбһ—бһ¶бһ–бһҖбһ¶бһҡбһ„бһ¶бһҡбһҡбһ”бһҹбҹӢбһңбһ·бһҹбҹ’бһңбһҖбһҡбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбһ‘бҹҒ бһҸбҹӮбһңбһ¶бһҗбҹӮбһҳбһ‘бһ¶бҹҶбһ„бһҹбһҳбҹ’бһӣбһ¶бһ”бҹӢбһўбһ¶бһҮбһёбһңбһҖбһҳбҹ’бһҳбһ‘бһ¶бҹҶбһ„бһҳбһјбһӣбһ•бһ„бһҠбҹӮбһҡ бһ–бҹ’бһҡбҹ„бҹҮбһҖбҹ’бһҡбһ»бһҳбһ бҹҠбһ»бһ“бһҠбһҫбһҡбһҷбһәбһҸбһҮбһ¶бһ„бһҠбҹғбһӮбһјбһ”бҹ’бһҡбһҖбһҪбһҸбһ”бҹ’бһҡбһҮбҹӮбһ„бҹ” бһңбһ·бһҹбҹ’бһңбһҖбһҡбһҠбҹҸбһҶбҹ’бһ“бһҫбһҳ бһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶бһўбҹ’бһ“бһҖбһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһ”бҹҶбһ•бһ»бһҸбһҠбҹӮбһӣбһӮбҹ’бһҳбһ¶бһ“бһўбҹ’бһ“бһҖбһҺбһ¶бһҷбһӣбҹӢбһ“бҹ„бҹҮбһ‘бҹҒ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһңбһ¶бһӮбһә **В«бһўбҹ’бһ“бһҖбһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠбһҠбҹҸбһҹбһ¶бһҳбһүбҹ’бһү бһҹбҹ’бһўбһ¶бһҸ бһ“бһ·бһ„бһҳбһ¶бһ“бһҡбһ”бҹҖбһ”бһҡбҹҖбһ”бһҡбһҷ бһҠбҹӮбһӣбһҳбһ“бһ»бһҹбҹ’бһҹбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһ“бһ¶бһҳбһҫбһӣбһҷбһӣбҹӢ бһ“бһ·бһ„бһўбһ¶бһ…бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ”бһ“бҹ’бһҸбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҡбһӣбһјбһ“бһ”бҹҶбһ•бһ»бһҸВ»**бҹ”

бһ…бһјбһҡбһ…бһ„бһ…бһ¶бҹҶбһҗбһ¶бҹ– **В«бһҖбһ»бҹҶбһҠбһҫбһҡбһ…бһјбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһјбһҠбһ…бһ¶бһҹбҹӢбҹ— бһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһҒбҹ’бһҹбҹӮбһўбҹҶбһ”бҹ„бҹҮ (Unit Tests) бһұбҹ’бһҷбһҹбҹ„бҹҮбҹ”В»**
