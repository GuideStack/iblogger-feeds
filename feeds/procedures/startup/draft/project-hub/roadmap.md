## Phase 1 — MVP Build
**Goal:** A working marketplace that supports one real transaction end-to-end.

- [ ] Buyer: register (email + OTP), browse, cart, checkout, order tracking
- [ ] Seller: register/verify, create store, add products, manage orders, payout
- [ ] Admin: approve sellers, manage categories, view orders
- [ ] Order flow + seller withdrawal flow
- [ ] Payment integration
- [ ] Core trust & safety (seller verification, reviews)

> *Scope = the must-have user stories only. Defer "optional" items (e.g. multiple stores per seller).*

## Phase 2 — Pilot Launch
**Goal:** Onboard a small set of real vendors + buyers and learn.

- [ ] Onboard first cohort of vendors (via [Partner Org] network)
- [ ] Soft launch to a limited buyer audience
- [ ] User onboarding & education materials (how to register, how to buy)
- [ ] Collect feedback; fix top issues
- [ ] Validate commission/revenue mechanics with real transactions

## Phase 3 — Growth
**Goal:** Grow both sides of the marketplace sustainably.

- [ ] Marketing & customer-acquisition campaigns
- [ ] Expand vendor base & product categories
- [ ] Analytics & KPI tracking (sales, active sellers, growth)
- [ ] Retention / loyalty features
- [ ] Iterate on revenue streams (promoted listings, premium features)
