# 🎓 Colleges — Curriculum Library

Subject-by-subject curriculum guides modeled on real US college programs. Organized by **college → major → subject area** so each major is clearly separated and new majors can be added easily.

---

## Structure

```
colleges/
└── <college-name>/
    └── <major-name>/
        ├── year_1.md ... year_4.md   ← year-by-year course plan
        └── <subject-area>/           ← deep-dive subject files
```
